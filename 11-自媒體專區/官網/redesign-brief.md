## 一、現況評估

### 現在的定位
❌ 「量化交易教學平台 + VIP 課程」  
→ 純技術向，易被其他教學頻道替代，難以建立個人品牌

### 改版後的定位
✅ 「高共感女性 × 系統思維 × 交易 × 心理覺察」的完整故事線

---

## 二、新定位與核心價值主張

### 品牌敘事

> **我是 Krystal，一個高共感的小資女。**
> 
> 我用系統思維從混亂人生裡奪回主導權。
> 通過認識自己、設定邊界、建立紀律，
> 我把資本複利成生命自由。
> 
> **理財交易，其實沒有你想的那麼難。**
> 
> 關鍵是找對自己的方式。

### 目標受眾細畫

| 特徵 | 描述 |
|------|------|
| **背景** | 不知如何理財 / 在關係裡迷失 / 想要自由但不知從何開始 |
| **痛點** | 害怕做錯決定 / 容易被情緒左右 / 想要系統但覺得太複雜 |
| **追求** | 財務獨立 / 情感健康 / 有掌控感的生活 |
| **年齡** | 25-40 歲 |
| **性別** | 主要女性（但不排斥男性讀者） |

### 核心價值 3 大支柱

1. **心理覺察優先** — 認識自己是一切的起點
2. **系統化思維** — 用紀律取代情緒
3. **行動導向** — 理論要落地成可執行的步驟

## 四、頁面設計方向

### 首頁 Hero Section

❌ 現在（功能列舉）
```
歡迎來到斜槓姐姐
- XScript 教學
- 看盤技巧
- 理財課程
```

✅ 改後（故事引入）
```
[大圖：Krystal 工作照或意象圖]

題目：「我花了十年，才發現自己的注意力從不屬於自己」

副題：一個高共感的小資女，如何用系統思維奪回生命主導權

[CTA 按鈕：「進來看我的故事」]
```

### 關於我頁面（新增）

**結構：**

1. **短故事（3-5 分鐘閱讀）**
   - 「我曾經是怎樣的」（諮商前：迷茫、被困在關係、不知道自己要什麼）
   - 「發生了什麼」（心理覺察、一個關鍵決定）
   - 「現在呢」（財務自由在望、心理更健康、找到了自己的聲音）

2. **核心三件事**
   - 為什麼我開始交易？
   - 為什麼我分享？
   - 我想幫助什麼樣的人？

3. **社會驗證（信任錨點）**
   - 2025 年度績效：YOY 94.18%
   - 管理資產：~500 萬
   - 2026 目標：1000 萬資產
   - 追蹤者數量 / 反饋

4. **我的方式 vs 別人**
   - 不是「一夜暴富」的故事（更誠實）
   - 不是「完美人生」的包裝（真實掙扎）
   - 而是「系統化 + 心理健康 + 行動」的結合

### 內容頁面改造

**從純粹教學文章 → 故事型教學**

例子：
- 舊：「XScript 語法教學 #1：如何寫一個簡單的策略」
- 新：「我如何用 XScript 把交易從情緒決策變成自動化紀律」

## 五、視覺與品牌語言

### 色系建議
- **主色**：深青/靛青（信任 + 專業 + 沈靜）
- **輔色**：金色或玫瑰金（溫暖、女性、成長）
- **背景**：淡灰或淡藍（不過度刺激）

### 排版建議
- 大量留白（減少視覺焦慮）
- 短句式標題（適合高共感讀者的認知習慣）
- 故事優先於數據（但數據要有）

### 字型
- 中文：思源黑體 / 思源宋體（清晰、專業、中性）
- 英文：Montserrat / Inter（現代、簡潔）

### 圖片 / 視覺元素
- ✅ 真實照片（Krystal 的工作照、咖啡館寫作、圖表）
- ✅ 手繪插圖（柔和、親切）
- ✅ 策略圖表（清晰視覺化交易邏輯）
- ❌ 過度修圖 / 虛假的「完美生活」圖

## 十一、預算與資源

### 若外包設計師
- 首頁設計：NT$10,000-15,000
- 全站 UI/UX：NT$25,000-40,000
- 內容遷移：自己做（或 NT$5,000-10,000）

### 若自己做（低成本）
- 使用 Webflow / Framer / Substack（託管解決方案）
- 花 2-3 週學習 + 製作
- 成本：Webflow 年費 $180 美元

### 工具推薦
- **網站建設**：Webflow（無程式碼、專業、易於維護）
- **郵件電子報**：Substack（簡單、免費起始）
- **SEO 追蹤**：Google Analytics 4（免費）

## 十二、常見問題

**Q：會不會洩露隱私？**  
A：不會。分享的是「普世的心理洞察」而非「具體發生了什麼」。所有案例匿名或概括化。

**Q：交易教學還重要嗎？**  
A：很重要，但優先度變成 2。現在是「先用故事引入 → 再教技術」而非「直接教技術」。

**Q：VIP 課程要下架嗎？**  
A：不用。但改為「加強班」而非「基礎」。基礎部分改為免費內容。

**Q：需要線上社群嗎？**  
A：未來考慮。現在先用 Threads + 電子報。
