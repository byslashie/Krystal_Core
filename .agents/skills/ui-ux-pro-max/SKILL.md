# UI-UX-PRO-MAX Skill

## 核心原則
1. **無障礙優先**：所有設計選擇必須優先考慮包容性存取。
2. **性能優化**：考慮加載狀態與資源效率。
3. **一致性**：維持整個應用程式的視覺和諧。
4. **設計美學**：追求極簡、現代且具衝擊力的視覺效果。

## 設計風格參考
- **Minimalist**: 簡約、留白、強調功能。
- **Glassmorphism**: 磨砂玻璃效果、層次感、柔和陰影。
- **Neumorphism**: 軟性 UI、與背景融合的凸起效果。
- **Material Design**: Google 官方設計規範。

## 色彩與字體
- 優先使用 HSL 色彩模式進行微調。
- 字體建議使用 Google Fonts (Inter, Roboto, Outfit, Noto Sans TC)。

## UX 準則
- 減少認知負荷：一次只呈現最重要的資訊。
- 回饋機制：所有操作必須有明確的視覺回饋。
- 移動優先：確保在各種螢幕尺寸下都能完美呈現。
