# XS-SKILL (XQ 腳本專家)

## 技能說明
此技能旨在強化對 XQ 全球贏家 (XQ Global Winner) 的 XS 語言撰寫能力。XS 是專為金融數據分析、策略回測與自動交易設計的腳本語言。

## 核心能力
1. **指標開發**：撰寫自定義技術指標 (Indicators)。
2. **警示策略**：開發即時盤中警示腳本 (Alerts)。
3. **選股策略**：編寫跨頻率、多條件的選股腳本 (Stock Picking)。
4. **回測優化**：優化策略參數，提高回測準確度。

## 語法規範
- 遵守 XS 語法結構（宣告、輸入參數、主邏輯）。
- 使用內建函數（如 `Value1`, `Condition1`, `average`, `rsi` 等）。
- 確保腳本在 XQ 平台中能正確編譯。

## 安裝注意
本技能依賴於外部資源（`xs_installer.zip`）。若要獲得完整能力，請確保該壓縮檔內的資源已整合至 Agent 的知識庫中。
