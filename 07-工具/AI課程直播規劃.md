## 🎯 核心定位

**「素人如何用 AI 打造自己的 XQ/XS 交易分析系統」**

- 不是教交易技術 → 是教 **AI 輔助交易決策的方法論**
- 核心賣點：**「以前寫 XS 要學程式，現在用 AI 5 分鐘生一個策略」**

## 📋 直播流程大綱（90 分鐘）

```
一個策略的誕生：從想法 → AI 討論 → XS 程式碼 → XQ 回測 → 結果分析
```

| 時段 | 段落 | 重點內容 |
|------|------|---------|
| 0–10 分 | 🎣 開場鉤子 | 先展示跑完的回測結果 → 製造「這是 AI 做的？」的驚喜感 |
| 10–25 分 | 💡 策略邏輯討論 | 用 AI 對話設計買進/賣出/停損條件，讓觀眾看到思考過程 |
| 25–50 分 | 🔥 LIVE DEMO | AI 生成 XS 程式碼 → 貼入 XQ → 跑回測，全程螢幕直播 |
| 50–65 分 | 📊 AI 分析結果 | 把回測數據貼給 AI，讓它指出問題並建議優化方向 |
| 65–75 分 | 🎁 送禮物互動 | 抽獎 / 關鍵字 / 填表換禮物 |
| 75–90 分 | ❓ Q&A | 開放提問，收尾呼籲 |

## 📄 禮物包內容細節

### ① XS 分析腳本
- 策略：KD 低檔翻揚 + 量能確認（或直播當天 DEMO 的策略）
- 格式：`.txt` 或 `.xqs` 檔案
- 包含：每一行附中文註解，新手也看得懂

### ② AI 配套 Prompt × 3

**Prompt A — 策略邏輯設計**
```
「我想設計一個台股策略，條件如下：[你的想法]
請幫我：
1. 分析這個邏輯的合理性
2. 建議加入哪些過濾條件
3. 用 XQ 的 XS 語法寫出程式碼，並附中文註解」
```

**Prompt B — 回測結果分析**
```
「這是我在 XQ 跑的回測結果：[貼數據]
請幫我分析：
1. 勝率和賠率是否合理？
2. 可能的問題在哪裡？
3. 建議如何調整參數或條件？」
```

**Prompt C — 週交易日誌 Review**
```
「以下是我本週的交易紀錄：[貼數據]
請幫我分析：
1. 虧損交易有什麼共同特徵？
2. 獲利交易有什麼模式？
3. 給我三個下週應改進的具體建議」
```
