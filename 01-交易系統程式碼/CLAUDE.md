## 📝 日誌和追蹤

### 文件命名規則
- 日誌：`[日期]_trading_log.json`
- 回測結果：`[日期]_backtest_[策略].csv`

### 必須記錄的信息

## 🎨 UI 顏色慣例（重要）

### 損益顏色：全站統一使用台股慣例

**全站一律用台股慣例**，不分台股 / 美股：

| 狀態 | 顏色 | CSS class |
| ---- | ---- | --------- |
| 盈利 / 上漲 | 🔴 紅色 | `class="green"` 或 `var(--green)` |
| 虧損 / 下跌 | 🟢 綠色 | `class="red"` 或 `var(--red)` |

> **實作方式：CSS 變數已對調** — `--green` 實際顯示紅色，`--red` 實際顯示綠色。
> 程式碼邏輯 `pnl >= 0 ? 'green' : 'red'` **不需要改**，自動顯示正確顏色。

### 注意事項

- 不要直接寫死 `color:red` 或 `color:green`，一律使用 `var(--green)` / `var(--red)`
- 系統狀態（連線 = 綠色圓點）等**非損益**語義顏色不受此規則影響
