## 🔧 常見問題

### ❓ Q1: 雙擊後什麼都沒有發生？

**可能原因**: Python 不在 PATH 環境變數中

**解決方案**:
1. 打開命令提示字元（Win+R，輸入 `cmd`）
2. 運行:
   ```bash
   python --version
   ```
3. 如果顯示版本號，Python 已安裝
4. 如果提示「找不到」，需要重新安裝 Python

### ❓ Q2: 出現「找不到 dashboard_v8」錯誤？

**可能原因**: .BAT 檔案位置不對

**解決方案**:
- 確保 .BAT 檔案在這個資料夾:
  ```
  G:\我的雲端硬碟\Krystal_完整系統\01-交易系統程式碼\
  ```

### ❓ Q3: 出現「埠口 5000 已被佔用」？

**可能原因**: 有其他程序使用同一埠口

**解決方案**:
1. 關閉其他 Flask 應用
2. 或者修改 .BAT 檔案，改用其他埠口:
   ```bash
   python -m http.server 8000  # 改為 8000
   ```

### ❓ Q4: 黑色命令窗口一閃而過？

**可能原因**: 發生錯誤

**解決方案**:
1. 使用「完整版」.BAT（會停留在窗口）
2. 或在命令提示字元中手動運行:
   ```bash
   cd "G:\我的雲端硬碟\Krystal_完整系統\01-交易系統程式碼"
   python -m http.server 5000
   ```

### ❓ Q5: 如何停止伺服器？

**解決方案**:
- 關閉黑色命令窗口（按 Alt+F4）
- 或在窗口中按 Ctrl+C

## 📝 高級配置

### 修改埠口

編輯 `.BAT` 檔案，找到這一行：
```batch
python -m http.server 5000
```

改為你想要的埠口（例如 8000）：
```batch
python -m http.server 8000
```

然後訪問: `http://localhost:8000/dashboard_v8/index.html`

### 自動刷新（開發模式）

如果你要修改 HTML，可以使用自動重新載入工具：
```bash
pip install watchdog
watchmedo shell-command -c "python -m http.server 5000" -d .
```
