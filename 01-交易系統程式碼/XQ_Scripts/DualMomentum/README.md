# 兩層 Dual Momentum 系統 for XQ / XS

架構：Layer1 族群篩選 → Layer2 個股排名 → Layer3 執行監控
作者：自用版本，每月月底跑一次

## 執行 SOP (每月底操作流程)

1. **Step 1 [月底 D-2]**
   - 跑 `DM_Layer1_SectorRank`
   - 候選池 = 族群代表股（上方 10 檔）
   - 記錄入選的前 3 族群

2. **Step 2 [月底 D-1]**
   - 跑 `DM_Layer2_StockRank`
   - 候選池 = 入選族群的所有成員（台股前100內）
   - 記錄每族群前 2 檔個股
   - → 共 6 檔為下月持股清單

3. **Step 3 [月初 D+1 開盤]**
   - 對照上月持股清單
   - 賣出「不在新清單」的持股（開盤價執行）
   - 買入「新進清單」的持股（等權重配置）

4. **Step 4 [月中監控]**
   - XS 每週自動跑 `DM_Layer3_Monitor`
   - 持股清單 = 目前持倉
   - 若出現「出場訊號」→ 紀錄，下月優先處理
   - 若出現「弱化警示」→ 留意，觀察是否持續惡化

## 重要參數設定建議
- **TopN（族群數）**: 建議 3，分散三個不同族群
- **TopStocks（個股）**: 建議每族群 2 檔，共 6 檔
- **MinLiquidity**: 建議 3億（台股前100通常遠超過）
- **弱化警示閾值**: 建議 score < 20
- **調整頻率**: 月底計算，月初第一個交易日執行
