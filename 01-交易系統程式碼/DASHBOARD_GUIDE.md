## 1️⃣ 📊 總覽（每天必看）

### 這頁在說什麼
把所有頁面的「最重要的一個數字」集中在同一個畫面，讓你 30 秒內掌握全局。

### 如何看

**第一排：宏觀訊號**
- 景氣象限標示（擴張/趨緩/衰退/復甦）— 來自總經羅盤
- PMI、CPI、非農就業人數 — 確認景氣方向沒有改變

**第二排：核心 KPI**
- 總資產：今天的總值，看有沒有異常波動
- 今日損益：正負號 + 金額，快速判斷今天的方向
- 持倉數：有沒有意外多出或少掉部位
- 風控狀態 ✅/⚠️/🚨：這個最重要，紅色代表要立刻去風控頁

**第三排：走勢 + 配置**
- 左圖（走勢）：今天的權益曲線是否延續趨勢
- 右圖（配置）：IB vs 元大 vs 現金的比例是否在計畫內

**第四排：持倉快覽 + 損益條圖**
- 前 5 筆持倉：確認最大部位的狀況
- 每日損益條圖：看近 20 天的節奏，判斷是連贏還是連虧期

**第五排：策略狀態 + 風控警示**
- 策略狀態：每個策略今天的損益，有沒有異常
- 風控警示：凱利倉位利用率、各風控指標進度條

### ⚡ 行動原則
| 看到 | 行動 |
|------|------|
| 風控狀態 🚨 | 立刻去⚠️風控頁，找出超標的策略 |
| 今日損益 < -1% 總資產 | 確認是哪個部位造成，考慮減倉 |
| 景氣象限改變 | 去💰資金配置頁，重新檢視配置比例 |
| 某策略損益異常 | 去📁策略管理頁，對比回測預期 |

## 2️⃣ 🌐 總經羅盤（財經M平方框架）

### 這頁在說什麼
用 MacroMicro × Hermit 的 5 步驟框架，判斷現在全球景氣在哪個象限，並給出對應的資產配置方向。

### 5 步驟如何看

**STEP 1：判斷全球經濟基本面**
- 看美國終端長循環：非農就業 + 零售銷售 + ISM製造業PMI
- 看海外製造短循環：中國先行指標 → 歐日台韓跟隨
- **判斷標準**：非農連續 3 個月下滑 = 經濟轉弱訊號

**STEP 2：判斷全球資金面**
- 看聯準會：政策利率方向 + 資產負債表規模
- 看通膨：CPI 年增率趨勢（上升/下降）
- 看實質利率：名目利率 - 通膨預期（正值 = 緊縮環境）
- **判斷標準**：Fed 轉向降息 = 資金面轉鬆，有利股市

**STEP 3：確認景氣循環落點（4象限羅盤）**

```
                物價上升
                    ↑
    趨緩            │            擴張
  Slow Growth      │           Growth
  (高殖利率/防禦)  │         (科技/非必需/工業)
                    │
經濟減速 ←─────────┼─────────→ 經濟加速
                    │
    衰退            │            復甦
  Recession        │           Recovery
  (現金/黃金/公債) │         (能源/工業/大宗商品)
                    │
                    ↓
                物價下降
```

**STEP 4：對應投資產業**
| 象限 | 資產配置方向 | 適合策略 |
|------|-------------|---------|
| 擴張 | 股票為主，挑長線成長資產 | 趨勢追蹤、動能策略 |
| 趨緩 | 高殖利率、防禦型、個別產業 | 均值回歸、股息策略 |
| 衰退 | 持有現金/黃金/公債，降低持股水位 | 空頭策略、對沖 |
| 復甦 | 提高風險資產，大宗商品、高收益債 | 逆勢建倉 |

**STEP 5：個股/ETF 判斷**
- 潛在市場：目標產業年複合成長率 > 10%
- 進入門檻：關鍵材料/技術/平台
- 競爭態勢：前三大企業市占 > 50%（寡占）或 > 50%（獨占）

### ⚡ 行動原則
- 景氣象限每季確認一次
- 象限改變時 → 去💰資金配置頁調整目標比例
- 象限在「衰退」時 → ⚠️風控頁把凱利係數調低到 0.5

## 3️⃣ 💰 資金配置（每月調整）

### 這頁在說什麼
根據景氣象限 + 凱利公式，決定各策略、各資產類別的最適配置比例。

### 如何看

**景氣象限連結**
- 頁面頂部顯示當前象限，並自動列出對應的「建議配置方向」
- 這是配置決策的最高指導原則

**實際 vs 建議配置圖**
- 橫條圖：綠色 = 在目標範圍內，黃色 = 需要調整，紅色 = 嚴重偏離
- 每條顯示：當前 % / 目標 %

**凱利公式計算器**
```
凱利公式：f* = (b × p - q) / b

輸入：
  p = 勝率（例：0.65）
  b = 平均獲利 / 平均虧損（例：2.0）
  q = 1 - p（敗率）

輸出：
  f* = 建議最大倉位佔比

範例：
  p = 0.65, b = 2.0, q = 0.35
  f* = (2 × 0.65 - 0.35) / 2 = 0.475 → 最多用 47.5% 資金
  實際建議用半凱利（f*/2）= 23.75%（更保守）
```

**景氣象限調整係數**
| 象限 | 凱利係數 | 說明 |
|------|---------|------|
| 擴張 | 1.0x | 全凱利 |
| 趨緩 | 0.7x | 保守 |
| 衰退 | 0.3x | 極保守 |
| 復甦 | 0.8x | 開始加碼 |

### ⚡ 行動原則
- 每月第一個交易日重新計算一次
- 配置偏離 > 5% 時才調整（避免過度交易）
- 凱利公式輸出的倉位是「上限」，不是「目標」

## 4️⃣ 💼 投資組合（每日確認）

### 這頁在說什麼
按券商分類顯示所有持倉的即時狀態，並提供走勢比較功能。

### 如何看

**持倉表（IB / 元大）**
- 損益% > +20%：考慮是否部分獲利了結
- 損益% < -8%：確認是否觸及停損線（你的規則：單筆 -2% 帳戶資金）
- 市值佔比：單一部位不超過帳戶 20%

**走勢比較圖**
- 藍線 = 我的投資組合（正規化報酬率）
- 其他線 = 比較標的（0050.TW、^TWII、SPY 等）
- 看法：我的線應該長期在大盤線上方，否則要重新檢討

**新增比較標的**
- 輸入任何 Yahoo Finance 支援的代碼
- 台股：`2330.TW` `0050.TW` `^TWII`
- 美股：`SPY` `QQQ` `NVDA`
- 自動抓取過去 12 個月月線資料

## 5️⃣ ⚠️ 風控（每日必看）

### 這頁在說什麼
監控整體組合 + 每個策略的風險指標，並用凱利公式計算每個策略的建議倉位上限。

### 如何看

**三大風控指標**
| 指標 | 正常 | 警戒 | 危險 |
|------|------|------|------|
| 最大回撤 | < -6% | -6% ~ -10% | > -10% |
| 年化波動率 | < 15% | 15% ~ 20% | > 20% |
| 夏普比率 | > 1.5 | 1.0 ~ 1.5 | < 1.0 |

**三條風控紅線（你的規則）**
- 單筆風險上限：帳戶 2%（超過要減倉）
- 每日風險上限：帳戶 5%（超過停止當天交易）
- 最大回撤限制：帳戶 10%（超過全面檢討策略）

**每策略風險明細**
- 看每個策略當前的風險佔比
- 確認沒有單一策略吃掉過多風險預算

### ⚡ 行動原則
- 任一紅線超過 80% → ⚠️ 警告，準備減倉
- 任一紅線超過 100% → 🚨 立即行動
- 夏普 < 1.0 連續 3 個月 → 重新評估該策略是否繼續

## 6️⃣ 💹 交易管理（交易時段使用）

### 這頁在說什麼
顯示所有進行中和已完成的實盤交易，對應到 Port 9999 的即時系統。

### 如何看

**系統連線狀態**
- IB / 元大 / Schwab 各自的連線狀態
- 任一顯示「斷線」→ 立刻檢查 API 連線

**當前開倉交易**
- 搜尋欄：快速找特定代碼
- 策略過濾：只看某個策略的部位
- 注意欄位：進場價 vs 現價的差距、持有天數

**已實現交易**
- 看平均損益%：應該對應回測的預期勝率
- 看持有天數：是否和策略設計的持倉期一致
- 連續虧損 > 3 筆 → 去📁策略管理頁確認策略狀態

## 7️⃣ 📁 策略管理（每週回顧）

### 這頁在說什麼
分兩個區塊：正在實盤運行的策略、還在研發測試的策略。

### 如何看

**區塊 A：實盤策略**

*實盤 vs 回測對比表*
| 指標 | 回測預期 | 實際表現 | 落差 |
|------|---------|---------|------|
| 年報酬率 | +24% | +18% | -6% |
| 勝率 | 65% | 58% | -7% |
| 最大回撤 | -8% | -12% | -4% |

- 落差 < 5%：正常，策略執行良好
- 落差 5~10%：注意，可能有滑點或市場環境改變
- 落差 > 10%：警示，策略可能失效，考慮暫停

*實盤 vs 回測 NAV 曲線對比圖*
- 兩條線應該走勢相近
- 實盤線持續低於回測線 → 可能是過度擬合（overfitting）問題
- 實盤線突然偏離 → 檢查是否有異常交易或市場結構改變

**區塊 B：研發策略**

*回測結果摘要*
- 顯示每個待上線策略的回測指標
- 通過上線標準才能移到「實盤策略」：
  - 年報酬 > 15%
  - 夏普 > 1.2
  - 最大回撤 < -15%
  - 勝率 > 55%
  - 樣本交易筆數 > 100

## 8️⃣ 📊 策略比對（每月分析）

### 這頁在說什麼
把所有策略的 NAV 曲線疊在同一張圖上，找出最強和最弱的策略。

### 如何看

**多策略 NAV 曲線圖**
- 所有線都從 100 出發（正規化）
- 最終值越高 = 該策略過去這段時間表現越好
- 看法：哪條線最穩定向上（不是最陡，是最平滑）

**回測 vs 實盤落差圖**
- 每個策略一個分組條圖
- 藍色 = 回測年報酬，橙色 = 實際年報酬
- 落差越小越好

**加碼/減碼建議**
- 根據凱利公式 + 近期表現，自動計算每個策略的建議配置
- 排行第一的策略 → 考慮加碼（在凱利上限內）
- 持續表現最差的策略 → 考慮暫停或減碼

## 9️⃣ 📈 績效（每月/季回顧）

### 這頁在說什麼
從整體帳戶層面檢視報酬，不是單一策略，是所有策略加總後的真實績效。

### 如何看

**4 大核心指標**
- 年度報酬率：目標 > 15%（超越大盤）
- 月均報酬率：目標 > 1.2%（年化約 15%）
- 勝率：目標 > 60%（有獲利的月份比例）
- 獲利因子：目標 > 1.5（總獲利 / 總虧損）

**月度報酬熱力表**
- 綠色月份越多越好
- 連續 2 個月虧損 → 重新檢視當時市場環境
- 月報酬 < -5% → 找出是哪個策略/部位造成的

**與大盤對比**
- 你的年報酬 vs SPY、^TWII
- 長期應該跑贏大盤，否則直接買 ETF 更省事

## 📐 凱利公式速查

```
完整凱利公式：
f* = (b × p - q) / b

b = 平均獲利 / 平均虧損
p = 勝率
q = 1 - p（敗率）

---

實際使用建議：

1. 先算出 f*
2. 乘以景氣係數（擴張1.0 / 趨緩0.7 / 衰退0.3 / 復甦0.8）
3. 取其 1/2（半凱利，更保守穩定）
4. 這才是你應該配置的倉位上限

---

範例計算：

趨勢追蹤策略：
  勝率 p = 0.65
  平均獲利 = 8%, 平均虧損 = 4%
  b = 8/4 = 2.0
  q = 1 - 0.65 = 0.35

  f* = (2 × 0.65 - 0.35) / 2 = 0.475

  當前景氣：擴張（係數 1.0）
  半凱利 = 0.475 × 1.0 / 2 = 23.75%

  → 這個策略最多用 23.75% 的帳戶資金
```
