## 🎯 新增功能詳細說明

### 頁面 5️⃣: 景氣循環與美林時鐘

#### 功能組成

**1. 美林時鐘 (Minsky Clock)**
```
四象限模型:
┌─────────────┬─────────────┐
│   復甦期 🔵 │   擴張期 🟢 │
├─────────────┼─────────────┤
│   衰退期 🔴 │   過熱期 🟠 │
└─────────────┴─────────────┘

坐標軸:
X軸: 通膨水平
Y軸: 經濟增長
```

**2. 關鍵經濟指標**
- 聯準會利率 (Fed Funds)
- 通膨率 (CPI)
- GDP 增長率
- 失業率
- 消費信心指數

**3. 景氣循環歷史追蹤**
- 12 個月的景氣狀態時間序列圖
- 相位變化趨勢分析

**4. 策略績效相關性分析**
- 不同景氣階段的策略表現對比
- 當前環境的推薦資金配置

#### 前端組件
- 側邊欄: 指標選擇、更新頻率、數據源配置
- 美林時鐘交互圖表 (Plotly)
- 經濟指標數據表
- 景氣追蹤折線圖
- 策略績效柱狀圖與圓餅圖

#### 數據來源 (待實現)
- FRED API (聯邦儲備經濟數據)
- Yahoo Finance API
- 自定義 API

#### 與後端整合 (待實現)
```
modules/econ_classifier.py 的 M1 輸出
    ↓
美林時鐘位置判斷
    ↓
Streamlit 頁面展示
    ↓
策略自動切換建議
```

### 頁面 6️⃣: 社群風險監控中心

#### 功能組成

**1. 實時風險評分卡**
- 整體風險評分 (0-100)
- 社群情緒指數
- 異常交易量統計
- 機構動向計數

**2. 實時風險告警 (過去 24 小時)**
- 嚴重程度標記 (🟢 低 / 🟡 中 / 🔴 高)
- 告警類型 (社群情緒、交易異常、機構動向、宏觀新聞)
- 事件描述與建議行動

**3. 風險指標 7 日趨勢**
- 整體風險評分折線圖
- 風險告警閾值標示 (可配置)
- 風險維度分析柱狀圖:
  - 社群情緒
  - 交易異常
  - 機構動向
  - 新聞事件
  - 技術面

**4. 社群信號詳細分析 (4 個 Tabs)**

**Tab 1: Twitter/X**
- 熱門標籤與提及次數
- 正面比例統計
- 情緒趨勢 (上升/下降/平穩)
- 熱門推文列表

**Tab 2: Reddit**
- 主要討論板塊
- 本週討論數量
- 熱門話題
- 整體情緒評估

**Tab 3: 金融新聞**
- 新聞來源
- 新聞標題
- 發佈時間
- 市場影響評估

**Tab 4: 機構動向**
- 主要機構名稱
- 最新投資動向
- 涉及資金規模
- 可能的市場影響

**5. 風險應對建議**
三列布局:
- 🛡️ 降低風險敞口
- ⚠️ 加強監控
- 📊 優化配置

#### 前端組件
- 側邊欄: 監控來源選擇、風險閾值設置、檢查頻率配置
- 4 個風險指標卡片 (Metric Cards)
- 告警列表 (可展開詳情)
- Plotly 折線圖 + 柱狀圖
- 四個選項卡式內容展示
- 建議與說明框

#### 數據來源 (待實現)
```
Twitter API / X API
    ↓
Reddit API
    ↓
金融新聞 API (News API, Bloomberg, Reuters)
    ↓
機構動向數據源 (SEC EDGAR, Bloomberg Terminal, etc.)
    ↓
多因子風險評分算法
    ↓
Streamlit 頁面展示
    ↓
Telegram/Email 自動告警
```

#### 與後端整合 (待實現)
```
modules/notifier.py
    ↓
風險告警推播 (Telegram / Email / Slack)
    ↓
決策引擎自動應對

交易系統風險限制
```

## 📝 使用說明

### 如何在 Streamlit 中查看新頁面

1. **確保 Streamlit 應用正在運行**
   ```bash
   cd "g:\我的雲端硬碟\Krystal_完整系統\01-交易系統程式碼"
   streamlit run app.py --server.port=5010
   ```

2. **刷新 localhost:5010**
   - Streamlit 應該自動檢測新頁面
   - 新頁面會出現在左側導航欄中

3. **點擊新頁面導航**
   - 🌍 景氣循環與美林時鐘
   - 🎯 社群風險監控中心

4. **測試交互功能**
   - 調整側邊欄參數
   - 與圖表交互
   - 查看不同的數據視圖
