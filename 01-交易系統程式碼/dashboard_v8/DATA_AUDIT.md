   - 已停用（`AUTO_DETECT_CLOSE = False`）
   - 根因：`append_trade` 欄位順序錯位 + broker normalize 不一致
   - 暫時可用，但**未修復**

2. **⚠️ trades sheet 損益欄幣別不一致**
   - 元大 = TWD，IB/Schwab/IBKR = USD
   - 已在計算端自動分類，但若新增其他券商會誤判
   - 建議：trades sheet 加一欄「幣別」明確標示

3. **⚠️ daily_nav_account 重跑時 Sheet header 擴充**
   - 已改 safe migration（永不 clear）
   - 但歷史「現金備檔」「緊急預備金」欄為空

4. **⚠️ 出入金沒記錄機制**
   - 5/07→5/08 市值跳升 60 萬大概率是入金，但 Sheet 顯示為「損益」-64K
   - 需建立 `cash_flow` sheet 並開始填

   - sync 時沒寫入 PNL
   - 影響：未實現勝率少算 4 檔

6. **⚠️ Schwab / 元大「現金」欄位無資料來源**
   - 顯示 `--`
   - 元大可手填、Schwab 需要 API
