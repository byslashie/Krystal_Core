## 槓桿計算

### 股票槓桿

**公式**：
```
槓桿倍數 = 1 / (1 - 保證金比例)
```

**範例**：
- 保證金比例 50% → 槓桿 = 1 / (1 - 0.5) = 2x
- 保證金比例 60% → 槓桿 = 1 / (1 - 0.6) = 2.5x
- 保證金比例 80% → 槓桿 = 1 / (1 - 0.8) = 5x

### 期貨槓桿

**公式**：
```
槓桿倍數 = 1 / 保證金乘數
```

**範例**：
- 保證金乘數 0.1 → 槓桿 = 1 / 0.1 = 10x
- 保證金乘數 0.05 → 槓桿 = 1 / 0.05 = 20x

## 交易成本

### 股票交易成本

**買入成本**：
```
買入總成本 = (成交價 × 股數) + 手續費 + 稅費
手續費 = 成交價 × 股數 × 手續費率 (台股: 0.001425%, 美股: 0.001% - 0.01%)
稅費 = 0 (買入時無稅費)
```

**賣出成本**：
```
賣出總成本 = (成交價 × 股數) - 手續費 - 稅費
手續費 = 成交價 × 股數 × 手續費率
稅費 = 成交價 × 股數 × 稅率 (台股: 0.3%, 美股: 0%)
```

**實際值 (台股元大)**：
```
買入手續費 = 成交價 × 股數 × 0.001425%
賣出手續費 = 成交價 × 股數 × 0.001425%
賣出稅費 = 成交價 × 股數 × 0.3%
```

### 期貨交易成本

**開倉成本**：
```
開倉成本 = 合約價 × 乘數 × 手續費率
```

**平倉成本**：
```
平倉成本 = 合約價 × 乘數 × 手續費率
```

## 損益計算

### 已實現損益 (Realized P&L)

**單筆交易**：
```
已實現P&L = (出場價 - 進場價) × 數量 - 手續費 - 稅費
已實現P&L% = (出場價 - 進場價) / 進場價 × 100%
```

**多個部位平倉**：
```
分段平倉P&L = Σ[(平倉價_i - 平均成本) × 數量_i] - 費用_i
```

### 未實現損益 (Unrealized P&L)

**當前持倉**：
```
未實現P&L = (現價 - 平均成本) × 持倉數量
未實現P&L% = (現價 - 平均成本) / 平均成本 × 100%
```

**多經紀商合併**：
```
總未實現P&L = Σ[(現價_i - 平均成本_i) × 持倉數量_i]
```

### 價格調整

**股票分割 (Stock Split)**：
```
調整後價格 = 原價格 × (新股數 / 舊股數)
調整後數量 = 原數量 × (新股數 / 舊股數)
```

**現金配息 (Dividend)**：
```
配息金額 = 每股股息 × 持股數量
成本基數調整 = 原成本 - 配息金額
```

## 回報計算

### 時間加權回報 (Time-Weighted Return)

**公式**：
```
TWR = ∏[(期末價值 - 期間現金流) / (期初價值 + 期間現金流)] - 1

分解步驟：
1. 分割現金流時間點
2. 計算各期回報率
3. 連乘各期回報率
4. 減 1 得到總回報
```

**範例**：
```
初始投入：$10,000
3個月後加入：$5,000

期1 (0-3個月)：
  期末價值：$12,000
  回報率：(12,000 - 0) / 10,000 = 0.2

期2 (3-6個月)：
  期末價值：$16,000
  回報率：(16,000 - 5,000) / 12,000 = 0.0917

TWR = (1.2 × 1.0917) - 1 = 31.00%
```

### 最大投資回報 (Max Invested Return)

**公式**：
```
MAX_ROI = (期末權益 - 累積投入) / 累積投入 × 100%

其中：
累積投入 = Σ所有進場投資額
期末權益 = 未平倉市值 + 已平倉收益
```

### 買入持有回報 (Buy & Hold Return)

**公式**：
```
BH_Return = (現價 - 進場價) / 進場價 × 100%
BH_Annualized = (1 + BH_Return) ^ (252 / 持有天數) - 1
```

### 指數對標回報

**相對回報**：
```
超額回報 = 策略回報 - 基準回報
超額回報% = (1 + 策略回報) / (1 + 基準回報) - 1
```

**基準選擇**：
- 台股：TAIEX (0050 或加權指數)
- 美股：S&P 500 (SPY)
- 混合：按投資比例加權

## 風險指標

### Sharpe 比率

**公式**：
```
Sharpe = (回報率 - 無風險利率) / 標準差

計算步驟：
1. 計算日回報率序列
2. 計算日回報率標準差
3. 年化標準差 = 日標準差 × √252
4. 年化回報率 = (期末 / 期初)^(252/日數) - 1
5. Sharpe = (年化回報 - 風險利率) / 年化標準差
```

**參數**：
- 無風險利率：2% (台灣) / 4% (美國)
- 計算頻率：日度
- 年化因子：√252 (交易日)

### Sortino 比率

**公式**：
```
Sortino = (回報率 - 無風險利率) / 下行標準差

其中：
下行標準差 = √[Σ(最小(回報率 - 目標收益, 0))^2 / N]
目標收益 = 通常設為無風險利率或 0%
```

### 最大回撤 (Maximum Drawdown)

**公式**：
```
MDD = (谷值 - 峰值) / 峰值

計算步驟：
1. 計算累積淨值曲線
2. 計算運行峰值 (running maximum)
3. 計算每日回撤 = (當日價值 - 運行峰值) / 運行峰值
4. MDD = 最小的每日回撤
```

### 恢復期 (Recovery Period)

**定義**：
```
從最大回撤谷值恢復到前期峰值所需的交易日數
```

### 勝率 (Win Rate)

**公式**：
```
勝率 = 獲利交易數 / 總交易數 × 100%
```

**詳細統計**：
```
連勝交易數 (Max Consecutive Wins)
連敗交易數 (Max Consecutive Losses)
平均獲利交易 (Avg Win)
平均虧損交易 (Avg Loss)
獲利因子 (Profit Factor) = 總獲利 / 總虧損
```

### Beta 係數

**公式**：
```
β = Cov(策略收益, 市場收益) / Var(市場收益)

計算步驟：
1. 取策略與基準相同期間的日收益率
2. 計算協方差
3. 計算市場方差
4. β = 協方差 / 方差
```

### Alpha 係數

**公式 (Capital Asset Pricing Model)**：
```
α = 策略回報 - [無風險利率 + β × (市場回報 - 無風險利率)]
```

## 期間分析

### 日度分析 (Daily)

```
計算單位：每個交易日
包含字段：
  - 日期
  - 開盤權益
  - 日P&L
  - 回報率
  - 持倉數
  - 成交量
  - 最高/最低權益
```

### 週度分析 (Weekly)

```
計算單位：每個自然週 (週一-週五)
聚合方式：
  - P&L：求和
  - 回報率：計算TWR
  - 勝率：按週統計
  - Sharpe：計算週標準差
```

### 月度分析 (Monthly)

```
計算單位：自然月
統計指標：
  - 月回報率
  - 月度Sharpe比率
  - 最大日回撤
  - 單月最佳/最差交易
  - 策略績效分佈
```

### 季度分析 (Quarterly)

```
計算單位：自然季 (Q1/Q2/Q3/Q4)
累積指標：
  - 季回報率
  - 季度Sharpe比率
  - 季度最大回撤
  - 策略貢獻度分析
```

### 年度分析 (Yearly)

```
計算單位：自然年
包含：
  - 年回報率
  - 年化Sharpe比率
  - 年度最大回撤
  - 完整績效報告卡
  - 與基準對標
```

## 圖表字段

### 權益曲線 (Equity Curve)

```
X 軸：時間 (日期)
Y 軸：累積權益 ($)
字段：
  - 權益 (實線)
  - 最高水位線 (High Water Mark，虛線)
  - 基準權益 (Benchmark，虛線)
```

### 日回報率分佈 (Daily Return Distribution)

```
X 軸：回報率區間 (%)
Y 軸：日數頻率
統計：
  - 正回報日數
  - 負回報日數
  - 平均日回報
  - 日回報標準差
```

### P&L 分佈 (P&L Distribution)

```
X 軸：P&L 區間 ($)
Y 軸：交易數頻率
統計：
  - 平均獲利交易
  - 平均虧損交易
  - 獲利因子 (Profit Factor)
  - 勝率
```

### 回撤曲線 (Drawdown Chart)

```
X 軸：時間 (日期)
Y 軸：回撤百分比 (%)
字段：
  - 當前回撤 (柱狀圖)
  - 最大回撤線 (目標線)
```

### 月度回報熱力圖 (Monthly Returns Heatmap)

```
X 軸：月份 (1-12)
Y 軸：年份
數值：該月回報率 (%)
顏色編碼：
  - 紅色：正回報
  - 綠色：負回報
  - 灰色：無交易
```

### 策略貢獻度 (Strategy Attribution)

```
圖表類型：派圖或堆積長條圖
維度：
  - 各策略的P&L貢獻
  - 各策略的回報率
  - 各策略的風險指標
```

## 實施注意事項

### 計算精度
- 貨幣：保留 2 位小數
- 百分比：保留 4 位小數 (e.g., 12.3456%)
- 日期時間：使用 ISO 8601 格式 (YYYY-MM-DD HH:MM:SS)

### 多幣種處理
- **台股 (TWD)**：使用元大持倉市值
- **美股 (USD)**：使用 Schwab/IB 持倉市值
- **合併報告**：使用實時 USD/TWD 匯率轉換 (daily close)

### 時區規則
- 台股：使用 Taiwan Time (UTC+8)
- 美股：使用 US Eastern Time，報告時轉換為 Taiwan Time
- 所有時間戳：統一轉換為 UTC+8

### 缺失數據處理
- **停牌股票**：使用最後成交價
- **退市股票**：使用清算價或最後成交價
- **匯率缺失**：使用前一交易日匯率
- **未知費用**：使用預估值並標記為 [ESTIMATED]

## 範例計算

### 完整交易範例

```
初始資本：$100,000
策略：RSI 均值回歸

交易 1：
  商品：2330.TW (台積電)
  方向：買入 (LONG)
  進場時間：2026-04-01 09:30:00
  進場價：450 元
  股數：100 股
  進場成本：45,000 × (1 + 0.001425%) = 45,000.64 元
  
  出場時間：2026-04-07 14:00:00
  出場價：465.50 元
  出場收入：46,550 × (1 - 0.001425% - 0.3%) = 46,411.39 元
  
  計算 P&L：
    毛利 = 46,550 - 45,000 = 1,550 元
    手續費 = (45,000 × 0.001425%) + (46,550 × 0.001425%) = 130.69 元
    稅費 = 46,550 × 0.3% = 139.65 元
    淨利 = 1,550 - 130.69 - 139.65 = 1,279.66 元
    回報率 = 1,279.66 / 45,000.64 = 2.84%

權益變化：
  Day 1 (4/1)：$100,000 → 進場後 $99,999.36 (扣進場費用)
  Day 6 (4/7)：$99,999.36 + $1,279.66 = $101,279.02
  
總投資回報率 = ($101,279.02 - $100,000) / $100,000 = 1.279%
```
