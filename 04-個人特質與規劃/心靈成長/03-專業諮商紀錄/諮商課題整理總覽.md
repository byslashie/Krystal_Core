## 📍 課題一：照顧者症候群（Caretaker Syndrome）

**本質**：將「承擔別人的課題」錯認為「愛」  
**狀態**：🔴 高度活躍

### 具體症狀
- 所有決策由自己做（約會地點、溝通、關係維持）
- 一直在「拉著別人前進」
- 即使精疲力竭還是持續這個模式
- 委屈但不敢表達，怕讓對方不舒服

### 根源
| 原生家庭觀察 | 內化的信念 |
|------------|----------|
| 母親為父親承擔所有家務情緒 | 「承擔 = 愛的表現」 |
| 父親冷漠退縮 | 「如果我不做，關係會破裂」 |
| 我在家扮演調停者 | 「照顧別人，才有資格被愛」 |

### 行動準則
- 看到混亂或他人擺爛時：深呼吸、轉身不介入
- 讓對方承擔他自己無能的後果（允許失敗發生）
- 每次衝動想接手時，問自己：**「這是他的課題，還是我的焦慮？」**

## 📍 課題二：安全感來源錯置（External Control = Security）

**本質**：把「掌控他人」當作獲得安全感的方式  
**狀態**：🔴 高度活躍（已有覺察，行為未完全改變）

### 具體症狀
- 他不照我的方式，就焦慮
- 焦慮時自動啟動掌控行為
- 知道不對，但停不下來（強迫症特徵）
- 需要他的反應來確認自己「有掌握狀況」

### 傳遞鏈
```
外婆/曾祖輩的創傷
    ↓
母親的焦慮型掌控（強迫症特徵）
    ↓
我：「掌控 = 安全，失控 = 危險」神經迴路
    ↓
Wei 越被控，越退縮（父親模式）
    ↓
他越退縮，我越焦慮掌控（惡性循環）
```

### 療癒方向
- 練習「安全感從內部生長」（不依賴外部確認）
- 感官錨點：回憶小籠包、肉燥飯→ 這些時刻**不需要掌控也能感到安全**
- 爵士樂的哲學：「沒有錯誤的音符」→ 允許失控，下一個音符會決定它是否正確

## 📍 課題三：自我價值感破洞（内部自信缺失）

**本質**：不相信自己「做自己就值得被愛」，需要不斷「有用」換取存在感  
**狀態**：🔴 根源級課題，所有問題的底層

### 具體症狀
- 容易在給我「被看見感」的人身上「心動」
- 伴侶習慣化後，那個感覺消失，去找新來源（出軌的底層機制）
- 害怕被忽視（同事沒打招呼就感到被拋棄）
- 從外部借來的自信（租來的自信）永遠不穩定

### 信念底層
> 「我必須先照顧別人、解決問題、有所貢獻，才有資格被愛。」

### 療癒方向
- 建立「內部自我價值感」：不依賴任何人的鏡射
- 辨別：「我真的愛這個人」 vs 「我愛『被他看見的自己』」
- 關鍵練習：**什麼都不做時，允許自己存在**（廢在旅館一整天也 OK）

## 📍 課題四：急迫性（Urgency Pattern）

**本質**：無法忍受「不確定」和「模糊」，強迫性地要求立刻得到答案  
**狀態**：🟠 正在處理中

### 具體症狀
- 立刻想知道：分手還是不分手
- 立刻需要看到他改變的證據
- 用數據分析十年對話，急著「下結論」
- 無法容許等待和觀察

### 根源
- 原生家庭老二的「快速反應者」角色
- 焦慮 → 行動（掌控）才能緩解焦慮
- 完美主義：結果必須確定，過程不能有模糊地帶

### 心理師的洞察
> 「妳越是急著找答案，妳就越陷入老模式（控制、推進度）。  
> 妳需要練習『不知道』和『等待』——這本身就是最大的成長機會。」

### 行動準則
- 每次想要立刻確認時：寫下感受，但不行動
- 訓練：在資訊不完整時，做得到「先觀察，再決定」嗎？
- **泰國旅行演練**：允許行程不完美，允許迷路，接受「不確定也很安全」

## 📍 課題五：界線模糊（Boundary Dissolution）

**本質**：把「他的課題」承擔到「我的課題」，角色從伴侶變成媽媽  
**狀態**：🟠 已有覺察，開始練習設立

### 承擔邊界侵犯清單
| 他的課題 | 我的行為 | 正確做法 |
|---------|---------|---------|
| 他的情緒 | 我安撫、我調節 | 「這是你的情緒，請你自己處理」 |
| 他的人生規劃 | 我代為決定約會、行程 | 讓他做決定，我等待而不催促 |
| 他的成長課題 | 我一直推他去諮詢 | 那是他的選擇，不是我的責任 |
| 他沒原諒媽媽的創傷 | 我替他難過、替他分析 | 放下，那是他的功課 |

### 近期進展 ✅
- 3/25 與媽媽的迷信衝突：用「課題分離」取代說教
- 3/26 第一次直接對媽媽說「這是有毒的觀念」
- 3/28 看穿媽媽的「抱怨遊戲」，沒有上鉤

### 下一步
- 繼續實踐「灰石策略」：只給無起伏的回應，不承接情緒垃圾
- 當他用 DARVO 攻擊時：**認出它，不被捲進去，不解釋**

## 📍 課題六：強迫性掌控（Compulsive Control）

**本質**：焦慮驅動的強迫性行為，以掌控他人來緩解焦慮  
**狀態**：🟡 已命名，待臨床評估

### 症狀檢核
- [x] 強迫性思考：反覆想「他為什麼不聽我」
- [x] 強迫性行為：反覆催促、主動聯繫、長篇說服
- [x] 知道不對但停不下來
- [x] 焦慮時自動掌控升級

### 待治療師確認
> 這是「臨床強迫症」還是「從母親學來的掌控行為」？

### 目前做法
- 認出衝動的瞬間：「我現在是在掌控他，還是在照顧自己的焦慮？」
- 讓衝動升起又消退，不立刻行動（衝浪，不淹沒）

## 📍 課題七：在關係中重演母親角色（Generational Pattern Reenactment）

**本質**：無意識地把母親對父親的互動模式，搬到自己對 Wei 的關係上  
**狀態**：🟡 已清晰看見，在練習打破

### 對照表
| 母親對父親 | 我對 Wei |
|----------|---------|
| 掌控、指揮 | 掌控、教導 |
| 否定他的想法 | 「我早說了」「再想想」 |
| 用關心名義控制 | 用「為你好」教導 |
| 焦慮時更掌控 | 他冷漠時我更推進 |
| 強迫性確認 | 反覆催促「什麼時候見面」 |

### Wei 對應的角色
Wei = 父親的重演：被控制 → 冷漠 → 退縮 → 放棄

### 療癒意義
> 不是在修復「我和 Wei 的關係」。  
> 而是在打斷「代際創傷的傳遞鏈」。  
> 即使最終分手，只要我改變了，這個循環就結束了。

## 📍 課題八：吸引逃避型伴侶的磁場（Complementary Attachment Trap）

**本質**：焦慮型依戀者（我）與逃避型依戀者（Wei）的互補吸引  
**狀態**：🟡 已理解，需要在下一段關係中主動改變

### 機制
```
我的焦慮型依戀 ←──────→ 他的逃避型
越追，他越逃             越逃，我越追
```

### 選擇未來伴侶的標準（已建立）
- 具備 **High Agency（高主動性）**，能自己定義問題、解決問題
- 對「我的大腦感到好奇」，不是感到壓力
- 是生活的「加速器」，不是「拖油瓶」
- 不需要我去「拉著他前進」

## ⚡ 當前最優先的三件事

> 根據諮商師 2026-04-22 的建議，優先級如下：

### 🥇 優先一：改變「急迫性」
**現在要練習的**：允許「不知道答案」。泰國旅行是絕佳演練場——迷路了不要焦慮，行程沒排好不要自責，廢一天也沒關係。

### 🥈 優先二：停止替他做決定
**現在要練習的**：下次他問要去哪/要怎麼辦，說「**我想聽你的想法，你決定**」——然後真的等，不催促，不提議，**不論等多久**。

### 🥉 優先三：讓身體休息
**現在要做的**：確保睡眠充足，繼續正面音頻（樊登），允許神經系統「不高速運轉」的時間。乾癬是身體在說「我需要停下來」。
