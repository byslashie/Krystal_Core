## 🎧 优先收听区：【樊登读书 APP】急救听书单 (简体字版)

因为目前处于急性戒断期（伴随肠胃紧缩、恶心、注意力无法集中的生理反应），直接看纸本书可能太耗大脑算力。**强烈建议前几天先用「樊登读书 APP」用听的**，让声音成为安抚副交感神经的平替对象，你可以一边做家事、休息甚至发呆时播放。

请在樊登读书 APP 中搜索以下书籍并**优先收听**（已依你现在的状况排列先后顺序）：

1.  **《被讨厌的勇气》（岸见一郎）**
    *   **核心对症（必听）：** **课题分离 (Separation of Tasks)**。这是你现在的终极解药。理解什么是「我的课题」，什么是「他的课题」。你太习惯把别人的课题扛在自己肩上，这本能帮你瞬间卸下对巨婴的罪恶感。
2.  **《亲密关系》（克里斯多福·孟）**
    *   **核心对症：** 为什么我们会被特定类型（冷漠冰山）的人吸引？因为我们在伴侣身上寻找原生家庭的镜像。这本书能完全呼应你关于「强迫性重复」与「渴望征服」的觉察。
3.  **《原生家庭：如何修补自己的性格缺陷》（苏珊·福沃德）**
    *   **核心对症：** 深度拆解「父爱匮乏」。了解你成为超级 PM 的原因，以及对「掌控权交接（M 倾向）」的深层极度渴望。
4.  **《情绪勒索》（苏珊·福沃德/周慕姿）**
    *   **核心对症：** 破解他的终极防御机制（DARVO）。当你听到他对你说「你就只想着你自己、都是你太强势」的时候，教你立刻听出这是勒索控制，斩断你的罪恶感迴路。
5.  **《非暴力沟通》（马歇尔·卢森堡）**
    *   **核心对症：** 戒掉「用不停做事来证明爱」的习惯。学习未来如何直接健康地表达需求，而不是用「过度承当」委屈自己。

## 第一階段：認識自己（讀這個先開始）

### 1. **《依戀理論與親密關係》**
**作者**：蘇珊·約翰遜 (Sue Johnson)
**中文出版**：《心理急救：建立穩定親密關係》或類似版本

**為什麼要讀**：
```
✓ 直接解釋焦慮型依恋是什麼
✓ 解釋「為什麼我需要掌控」
✓ 解釋「為什麼當他冷漠時我更焦慮」
✓ 提供改變依恋模式的方法
✓ 談論伴侶間的「追逐-逃跑」循環（你和Wei現在就在這個循環中）

關鍵章節：
- 焦慮型依恋的特徵
- 安全依恋如何建立
- 親密關係中的脆弱性
```

**閱讀時間**：2-3週

---

### 2. **《關係黑洞：走出共依存的陷阱》**
**作者**：哈麗特·勒納 (Harriet Lerner) 或 梅瑞狄斯·貝克 (Meredith Blake)

**為什麼要讀**：
```
✓ 直接定義「共依存」
✓ 解釋你為什麼用他人的反應定義自我價值
✓ 說明「幫助他」實際上是「掌控他」
✓ 解釋為什麼你無法放手
✓ 給出具體的改變步驟

關鍵內容：
- 共依存的10個特徵（你可能全部中招）
- 為什麼聰明人也會陷入共依存
- 如何恢復自我
- 健康關係vs.共依存關係的區別
```

**閱讀時間**：2-3週

---

### 3. **《完美的謊言：控制與自由的真相》**

**為什麼要讀**：

```
✓ 給出停止掌控的策略

關鍵內容：
- 掌控者的心理特徵
- 掌控如何摧毀關係
- 如何識別掌控行為
- 改變掌控的具體步驟
```

**閱讀時間**：1-2週

## 第五階段：能量運作系統與過度承擔 (High vs Low Agency)

### 10. **《情緒勒索》與《為什麼我們總是在逃避？》**
**作者**：蘇珊·福沃德 (Susan Forward) / 岡田尊司
**為什麼要讀**：
✓ 解析 Low Agency (低主動性) 人格如何利用「武器化無能」來控制伴侶
✓ 徹底搞懂 DARVO（否認、反擊、受害者角色互換）的運作機制
✓ 幫助你放下「他是不是因為太愛我/受傷才這樣」的幻覺，看清這是純粹的能量吸血

### 11. **《即使有罪惡感，也要劃清界線》(Boundary Boss)**
**作者**：泰瑞·柯爾 (Terri Cole)
**為什麼要讀**：
✓ 專門寫給 High Agency (高能動性 / 超級 PM) 的過勞人格
✓ 徹底剖析「過度承擔 (Over-functioning)」與「功能低下 (Under-functioning)」的病態共生
✓ 解釋為什麼「停止幫忙」會引發巨大的罪惡感，以及如何熬過戒斷期

**閱讀時間**：2週
