# 💰 Krystal 自媒體內容戰略

## 🎯 核心定位

**財經 KOL / 交易系統教練 / 人生規劃設計師**

> 目標 TA：30-40 歲的知識工作者，想要在交易 + 職涯 + 健康間找到平衡

---

## 📊 三大內容支柱

### 支柱 1️⃣ 交易系統分享（40% 內容量）

#### 核心定位
- **主題**：台指期策略、籌碼分析、技術指標、風險管理
- **TA 痛點**：
  - 「怎麼穩定獲利？」（持續性盈利）
  - 「怎麼管理風險？」（心理素質 + 系統性）
  - 「為什麼我的策略總是虧？」（驗證不足）

#### 內容形式

| 頻率 | 形式 | 內容例 |
|---|---|---|
| 日 | 實盤日誌 | 進場訊號 + 出場理由 + 當日績效 |
| 週 | 籌碼分析 | 外資/投信動向 + 持股變化 + 週操作複盤 |
| 週 | 技術分析 | K 線型態 + 均線系統 + 支撐阻力 |
| 月 | 策略績效回顧 | 勝率、平均收益、最大虧損、改進方向 |
| 月 | 指標研究 | 新指標測試、舊指標優化、效果驗證 |

#### 變現路徑

| 階段 | 時間 | 形式 | 預期收入 |
|---|---|---|---|
| **1. 內容積累** | 2026.5-2026.9 | 免費文章 + 社群 | - |
| **2. 知識沉澱** | 2026.10-2026.12 | 免費策略模板、成交記錄 | 贊助、廣告 |
| **3. 課程化** | 2027.Q1 | 「台指期 30 天上手」課程 | NT$3,000-5,000/人 |
| **4. 深度培訓** | 2027.Q2+ | VIP 諮詢、實盤陪伴 | NT$10,000+/月 |

#### ⚠️ 何時 NOT 做課程

```
❌ 2026 年做課程
   理由：策略還在驗證期（v1.1 紙面交易中），沒有足夠數據

❌ 沒有 100+ 個實盤交易記錄就宣傳「教人交易」
   理由：沒有說服力，反而傷害信用

❌ 沒有 60%+ 勝率就賣課
   理由：學生會虧錢，退課率高，負面評價

✅ 2027.Q1 以後再考慮課程化
   理由：到時候已有 6 個月的實盤數據 + 驗證結果
```

### 支柱 2️⃣ 人生規劃實錄（40% 內容量）

#### 核心定位
- **主題**：四重身份時間管理、決策框架、PDCA 迭代、實驗設計
- **TA 痛點**：
  - 「怎麼在多個身份間平衡？」（時間壓力）
  - 「怎麼快速驗證想法？」（決策效率）
  - 「為什麼我總是累？」（注意力管理）
  - 「如何從失敗中恢復？」（韌性建設）

#### 內容形式

| 頻率 | 形式 | 內容例 |
|---|---|---|
| 週 | 週反思文章 | 本週亮點 + 教訓 + 下週重點 |
| 週 | 決策案例分析 | 「我怎麼砍掉了一個無效投資」 |
| 月 | 月度目標檢查 | 目標完成度、調整策略 |
| 月 | PDCA 實驗複盤 | 計劃 → 執行 → 檢查 → 調整 |
| 季 | 身份檢查 | 交易績效 + PM 進度 + 自媒進度 + 健康指標 |

#### 變現路徑

| 階段 | 時間 | 形式 | 預期收入 |
|---|---|---|---|
| **1. 內容積累** | 2026.5-2026.12 | 免費週反思、月檢查 | - |
| **2. 社群沉澱** | 2026.11-2026.12 | 讀者互動、FAQ 積累 | 讚賞、小額贊助 |
| **3. 線上社群** | 2026.Q4 | Discord/Circle 付費社群 | NT$500-1,000/月/人 |
| **4. 諮詢服務** | 2027.Q2+ | 1:1 生涯規劃諮詢 | NT$5,000-10,000/次 |

#### ⚠️ 何時 NOT 做線上課程

```
❌ 在「人生規劃」這個支柱上做系統課程
   理由：我還在驗證自己的框架（2026 年內），
        教人不成熟的系統有風險

✅ 線上社群（Discord/Circle）可以做
   理由：這是「陪伴式」而非「系統式」，邊做邊優化

✅ 1:1 諮詢可以做（2027.Q2+）
   理由：經過 12 個月實驗，框架已驗證
```

### 支柱 3️⃣ 身心平衡實驗（20% 內容量）

#### 核心定位
- **主題**：低 FODMAP 飲食、迷走神經激活、自律神經調節、健康實驗設計
- **TA 痛點**：
  - 「我有腸胃問題，怎麼辦？」（SIBO / IBS / 脹氣）
  - 「為什麼我總是疲勞？」（自律神經失衡）
  - 「怎麼改善睡眠？」（恢復力）
  - 「有沒有科學驗證的方案？」（信任度）

#### 內容形式

| 頻率 | 形式 | 內容例 |
|---|---|---|
| 週 | 4 週驗證日誌 | 飲食記錄 + 症狀評分 + 數據對比 |
| 月 | 實驗複盤 | 「低 FODMAP 4 週效果評估」|
| 月 | 技巧分享 | 「迷走神經激活的 3 種簡單方法」 |
| 季 | 健康績效統計 | 能量等級、消化狀態、睡眠品質 |

#### 變現路徑

| 階段 | 時間 | 形式 | 預期收入 |
|---|---|---|---|
| **1. 內容積累** | 2026.5-2026.9 | 免費實驗日誌 + 技巧文章 | - |
| **2. 驗證沉澱** | 2026.10-2026.12 | 「我的健康實驗」電子書 | NT$300-500 |
| **3. 社群沉澱** | 2026.Q4 | 健康實驗社群（免費參與） | 贊賞、廣告 |
| **4. 商品銷售** | 2027.Q1+ | 食材清單 + 推薦商品聯盟 | 聯盟佣金 |

#### ⚠️ 何時 NOT 做課程

```
❌ 在自己還在實驗期就教人「健康方案」
   理由：效果還沒完全驗證，責任重大

❌ 推銷「健康產品」在驗證前期
   理由：沒有個人成功案例，銷售力弱

✅ 分享「實驗過程」，不是「推銷答案」
   理由：誠實透明，TA 會信任

✅ 電子書（2026.12）+ 社群（2026.Q4）
   理由：門檻低，有人會自然購買
```

## 👥 TA 分析 - 開放式編碼

### TA 人物像

```
年齡：30-42 歲
職業：軟體/產品經理、投資人、自由工作者
收入：月收 NT$80k-300k
痛點群：
  ├─ 交易相關
  │  ├─ 想穩定獲利，但總是虧錢
  │  ├─ 害怕「高手割韭菜」，想學真實系統
  │  └─ 需要有實盤驗證的策略教師
  │
  ├─ 人生規劃相關
  │  ├─ 工作壓力大，想要平衡多個身份
  │  ├─ 決策效率低，總是在瞎忙
  │  └─ 需要「實錄式」的指導，不要理論
  │
  └─ 健康相關
     ├─ 長期疲勞、消化不佳
     ├─ 不信任複雜的醫學診斷
     └─ 想要「低成本、快速驗證」的方案
```

### 核心驅動力

| 驅動力 | 表現 | 內容應對 |
|---|---|---|
| **信任度** | 「有沒有真的成功案例？」 | 分享實盤記錄、實驗日誌 |
| **效率** | 「我時間有限，要快速見效」 | 4 週驗證、3 秒決策框架 |
| **透明度** | 「不要虛構的故事，要真實數據」 | 完整日誌、失敗也記錄 |
| **可操作性** | 「我要能立即執行」 | 具體步驟、工具清單 |

## 📈 內容日曆框架

### 月度內容規劃

```
週一～週四：交易系統分享（4 篇）
  - 週一：實盤交易日誌
  - 週二：技術分析
  - 週三：籌碼分析
  - 週四：策略複盤

週五：人生規劃（1 篇）
  - 週反思 / 決策案例 / 目標進度

週六日：身心平衡（1-2 篇）
  - 實驗日誌 / 技巧分享 / 健康績效

月底（如有）：月度總結（1 篇）
  - 三個支柱的月績效統計
```

### 內容產出時段

```
19:30-21:00（奇週）→ 內容創作（2-3 篇）
19:30-21:30（偶週）→ 交易策略回顧（準備下週內容素材）
週六 14:00-18:00   → 內容深度製作（電子書、長文、視頻腳本）
週日 14:00-17:00   → 社群管理 + 數據分析
```

## 🚀 平台策略

### 主平台：個人部落格 / Substack

- **優勢**：完全掌控、讀者資料擁有、變現靈活
- **內容**：所有三支柱的長文、實驗日誌
- **頻率**：每週 5-6 篇

### 副平台 1：LinkedIn / Threads

- **優勢**：職業人脈、自媒體生態
- **內容**：交易日誌摘要、人生規劃洞察
- **頻率**：每週 3-5 則

### 副平台 2：YouTube（未來考慮）

- **優勢**：長視頻、教育內容、演算法推薦
- **內容**：策略講解、人生規劃分享、實驗複盤
- **頻率**：每週 1 個視頻（2026.Q4 後考慮）

### 副平台 3：Discord / Circle 社群（2026.Q4）

- **優勢**：高互動、付費轉化、社群網絡
- **內容**：每日交易訊號、實時問答、月度工作坊
- **會費**：NT$500-1,000/月

## ⚠️ 禁止行為

```
❌ 在驗證期間（2026 年內）就推銷課程
   → 沒有足夠數據，傷害信用

❌ 分享「未驗證的建議」
   → 所有內容必須基於實驗數據或學術文獻

❌ 同時推進 3 個以上的變現管道
   → 只關注「賺錢」而忽視內容質量

❌ 發佈「成功故事」而隱藏「失敗案例」
   → TA 價值觀是「透明度」，必須 balance

❌ 用自媒體賺錢 > 用自媒體學習
   → 優先級是「內容質量 > 收入」
```
