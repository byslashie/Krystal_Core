# Mini Gap 啟動計畫（v1.2 30 歲校準）

> [!QUOTE]
> Mini gap 不是「等到很有錢才能做的事」。
> 是「**用現有條件，定期練習自由**」。
> 一次比一次深，一次比一次清楚自己要什麼。

> [!IMPORTANT] **2026-05-13 v1.2 重大變更**
> 你今天告訴我「**30 歲了，想要的不能再等**」+「**12 月想去越南獨旅**」+「**想凍卵**」+「**現有關係不確定（6 個月 deadline）**」。
>
> 規劃因此大幅前移：
> - **2026 Q4 立刻加入越南獨旅 5-7 天 ⭐**（不等 2027）
> - 階段命名從「年份」改為「**年齡**」（每次看都會被提醒）
> - Mini gap 不只是放鬆，是「**人生決策的儀式**」
> - 整合凍卵 + 關係 deadline 到整個時程

---

## 起點現實（2026-05-13）

| 項目 | 現況 |
|---|---|
| 身份 | 嘉實資訊 量化交易 PM（朝九晚五，一週 1 天可遠端） |
| 特休 | 15 天/年，可全修 |
| **2026 剩餘特休** | **5 天**（已用 10 天：跟媽媽 + 朋友出國） |
| 留停制度 | ✅ 公司有 |
| 對工作感受 | A — 喜歡，只是想偶爾喘息 |
| 預算可動用 | 資產 500 萬，每次 mini gap 10-20 萬完全 cover |
| 居住成本 | 0（住家裡） |

### 關鍵洞察：你不缺特休天數，你缺「**真正照顧自己的特休**」

> 已用的 10 天：跟媽媽、朋友出國——「**沒有真正照顧到自己**」
>
> 這不是壞事，但它告訴我們：
> - 你會用特休，但都用在「**關係維護**」
> - 對你來說，「**獨處 / 一個人的時間**」可能是被忽略的需求
> - 第一次真正的 mini gap，核心要求是「**獨旅**」⭐

## 四階段架構（v1.2 — 30 歲版）

### 階段 0：**2026 Q4（30 歲）— 越南獨旅 5-7 天 ⭐**

> [!IMPORTANT] **這不再是「不勉強保留彈性」，是「30 歲送給自己的第一份禮物」。**

**核心策略**：**今年內必須做，不等 2027**

```
時程：2026 年 11-12 月
地點：越南（你已決定）⭐
天數：5-7 天（用剩 5 天特休 + 週末）
預算：3-6 萬（越南成本低）
```

#### 為什麼是越南（且為什麼是現在）

| 為什麼越南 | 為什麼現在 |
|---|---|
| ✅ 飛行短（3.5h，不浪費假期） | ✅ 你已用 10 天特休照顧別人，這 5 天該給自己 |
| ✅ 成本低（5-7 天 3-6 萬可結束） | ✅ 30 歲是「想要的不能再等」的年紀 |
| ✅ 第一次獨旅低門檻 | ✅ 凍卵諮詢 + 關係釐清都需要思考時間 |
| ✅ 咖啡文化 + 慢生活適合內省 | ✅ 越南獨旅 = 沉澱所有重大決策的儀式 |

#### 候選城市（越南）

| 城市 | 適合度 | 主題 |
|---|---|---|
| **大叻 Đà Lạt** ⭐ | ⭐⭐⭐⭐⭐ | 高原氣候、咖啡園、慢活、適合內省 |
| **會安 Hội An** | ⭐⭐⭐⭐ | 古城、燈籠、適合走路、攝影、安靜 |
| **峴港 Đà Nẵng** | ⭐⭐⭐ | 海邊、現代化、適合放鬆 |
| **胡志明市** | ⭐⭐ | 大都市、噪音多、不適合第一次獨旅 |
| **河內** | ⭐⭐⭐ | 文化深度、但比較吵雜 |

> [!TIP] **推薦組合：大叻 4 天 + 會安 2 天**
> - 大叻：高原咖啡 + 寫日記 + 走路 + 想事情
> - 會安：古城 + 攝影 + 慢慢吃 + 收尾

#### 這 5-7 天的「真實任務」

不是觀光，是**完成三個決策思考**：

1. **凍卵決策**（用一天全部想清楚）
   - 我為什麼要凍 / 不凍？
   - 30-50 萬對我來說的意義？
   - 回國後第一個月內預約諮詢

2. **關係決策**（這趟旅行的核心）
   - 6 個月 deadline 倒數開始
   - 不帶情緒地問：「他是不是合適？」
   - 用 Vision Board 那 8 條 criteria 一條條對

3. **2027 大方向**（順便）
   - 2027 Q1 第二次獨旅去哪
   - 凍卵時程怎麼安排
   - 留停試水的可能性

#### 核心要求（不可妥協）⭐

1. **一個人去**：不帶男友、不帶媽媽、不帶朋友
2. **不發 IG 限動**：這趟不是給別人看
3. **每天至少 4 小時獨處**：不視訊、不抓著聯絡
4. **帶一本筆記本實體寫**：不在手機打字
5. **手機公司 Email 通知關掉**

#### 30 歲第一次獨旅的心理準備

> [!NOTE]
> 你 30 年都跟家人 / 朋友 / 男友一起旅行。
> 第一次自己一個人，可能會：
>
> - 第 1-2 天：不習慣、想找人說話、覺得無聊
> - 第 3-4 天：開始享受、發現自己其實 OK
> - 第 5-7 天：「**這就是我**」的存在感回來了
>
> **不舒服是正常的**。不舒服就是練習的本質。

> 凍卵前的最後一次完整自由 + 凍卵後的能量充電

**前提**：階段 0（越南）完成，且體驗良好

### 進階獨旅（原階段 1 - 31 歲版）

> [!IMPORTANT] **這次比越南更深、且要考慮凍卵時程。**
> 凍卵建議在 Q2-Q3，所以這次獨旅放在 **Q1**（凍卵前的能量充電）

#### 觸發條件

```
□ 2027/1 月特休歸零重算 → 拿到新的 15 天
□ 公司業務不在高峰期（避開季報、年底結算）
□ 副業現金流穩（即使請假 19-21 天，0050 自動扣款不受影響）
□ 心理準備好：「我願意一個人 19-21 天」
□ 跟主管打過招呼（不必說 mini gap，說「想把特休一次用完」即可）
```

#### 時程設計（建議：農曆年後 / 3-4 月）

```
方案 A（推薦）：3 月中 - 4 月初
  - 跨美股財報季尾、台股年報季尾
  - 春天，天氣舒適
  - 飛機票 / 住宿不是旺季價格

方案 B：農曆年後（2 月中-3 月初）
  - 接著農曆年連假更划算
  - 但回來會接到積壓工作

請假方式：
  特休 15 天 + 前後週末 + 公司 1 天遠端 = 19-21 天連續
```

#### 地點建議（量化 PM × 獨旅 × 第一次）

| 候選 | 飛行時間 | 預算/月 | 適合理由 |
|---|---|---|---|
| **京都** ⭐ | 3.5h | 12-18 萬/3週 | 寧靜、適合內省、文化深度、獨旅友善 |
| **清邁** | 4h | 6-10 萬/3週 | 低成本、長住社群、digital nomad 友善 |
| **沖繩** | 1.5h | 10-15 萬/3週 | 飛行短、自然療癒、第一次獨旅低門檻 |
| **首爾** | 2.5h | 12-16 萬/3週 | 美食購物、文化、容易適應 |
| **峇里島烏布** | 5h | 8-12 萬/3週 | 瑜伽 / 冥想 / 療癒，最適合「照顧自己」 |

> [!TIP] **第一次的選擇邏輯：**
> 別選你「**會跟朋友/媽媽一起去**」的地方（會啟動慣性社交模式）。
> 選你「**只屬於自己**」的地方——一個你想像中「**獨自坐在咖啡廳寫日記**」的城市。

#### 工作模式（mini gap 期間）

```
週一-五（一週 1 天遠端可正當化）：
  - 每天上午 2-3 小時：交易系統檢查、Email、必要會議
  - 其他時間：休息、探索、寫作、運動

週末：
  - 完全 off，可去當地深度體驗

⚠️ 公司只允許「一週 1 天遠端」 → 19-21 天裡的「上班日」全部請特休
   不要混淆「特休」與「遠端」，避免麻煩
```

#### 核心要求（不可妥協）⭐

1. **一個人去**：不帶媽媽、不帶朋友、不帶男友
2. **不安排太多行程**：留白比塞滿重要
3. **每天獨處 ≥ 4 小時**：不見人、不視訊、不抓緊聯絡
4. **寫日記**：紀錄「沒有他人需求時，你想做什麼」
5. **不工作就完全不工作**：休息時段就真的休息

#### 預算估算（京都 19 天版本）

| 項目 | 金額 |
|---|---|
| 機票（含稅） | 1.5-2 萬 |
| 月租公寓（19 天） | 5-7 萬 |
| 飲食（中等水準） | 3-4 萬 |
| 交通 + 體驗 | 1.5-2 萬 |
| 緩衝 | 1 萬 |
| **合計** | **12-16 萬** |

> 對 500 萬資產 = **3% 以下**，完全可承受。

#### 階段 1 完成後的自問清單

回國後一週內，誠實回答：

- [ ] 「我一個人 19-21 天，感覺如何？」
- [ ] 「沒有他人需求時，我真的想做什麼？」
- [ ] 「想念正職工作嗎？想念哪部分？」
- [ ] 「想念台灣嗎？想念什麼？」
- [ ] 「如果我可以一年做 3 次這樣的旅居，會想嗎？」
- [ ] 「如果我可以全年這樣生活，會想嗎？」

**答案決定階段 2 怎麼走**。

### 階段 2：**2027 Q4 / 2028，留職停薪 1-3 個月**

#### 觸發條件

```
□ 階段 1 完成，且體驗良好
□ 副業現金流 > 月支出 50%（不依賴正職也能過半）
□ 資產達 800-1000 萬（中性情境 2027 末 ~830 萬，2028 末 ~1060 萬）
□ 跟主管/HR 溝通過留停可能性
□ 心理準備好「3 個月不上班」的真實感
```

#### 期間建議

```
方案 A：1 個月（保守）
  - 試水「沒有正職」的真實感
  - 風險低、回去容易

方案 B：3 個月（推薦）
  - 真正體驗 lifestyle
  - 留停 1 個月不夠看出長期效應
  - 但要先用方案 A 試過再做

時機：避開公司關鍵專案
  - Q4：年底結算前
  - Q1 結束後：3-5 月相對淡季
```

#### 地點策略

```
不再「換地方旅遊」，而是「找一個 base 深度住」

選一個階段 1 試過最愛的城市
住長租公寓（不是 Airbnb 短租）
建立 daily routine：
  - 固定健身房 / 咖啡廳 / 市場
  - 認識 1-2 個當地連結
  - 感受「住下來」不是「過客」
```

#### 工作模式

```
本業：完全停（這就是留停的意義）
副業：每天 2-4 小時
  - 交易系統運作 + 學習
  - 寫作（抒發型自媒體可在這階段啟動）
  - 策略研究 / 量化模組精進
休閒：每天 4-6 小時
  - 運動 / 冥想 / 閱讀 / 旅行
```

#### 預算估算（3 個月清邁版）

| 項目 | 金額 |
|---|---|
| 機票來回 | 2 萬 |
| 月租公寓 × 3 | 9-12 萬 |
| 飲食 × 3 月 | 6-9 萬 |
| 健身房 / 體驗 / 交通 | 3-4 萬 |
| 緩衝 + 簽證等雜支 | 2 萬 |
| **合計** | **22-29 萬** |

> 對 800-1000 萬資產 = **3% 以下**，仍完全可承受。

#### 階段 2 完成後的關鍵問題

```
3 個月後，你會非常清楚：
  → 「我想回嘉實上班」（A1）
  → 「我想辭職全力自由」（A2）
  → 「我想轉兼職 / 顧問」（A3）

這個答案不是用「想」出來的，是用「體驗」過才知道的。
```

### 階段 3：**2029-2030，真正的選擇點**

#### 不在這份計畫裡決定

> 階段 3 的決定，**取決於階段 1 + 階段 2 的真實體驗**。
> 現在的你還無法知道答案，**3 個月留停後的你**比現在的你更清楚。

#### 三條可能路徑

```
路徑 A1：穩定派（回嘉實）
  - 繼續上班，享受嘉實量化環境
  - 一年 2-3 次 mini gap 維持自由感
  - 資產自然累積到 3000 萬
  - 適合：留停後發現「其實還是想上班，只是想有彈性」

路徑 A2：自由派（辭職）
  - 全力做交易 + 抒發創作
  - 收入降低但 lifestyle 大幅提升
  - 資產累積較慢但時間自由
  - 適合：留停後發現「我已經不想再回辦公室了」

路徑 A3：中間派（轉兼職 / 顧問）
  - 嘉實兼職 / Part-time 顧問
  - 保留量化圈連結 + 部分收入
  - 大量時間做自己的事
  - 適合：留停後發現「想保留一些連結，但不要 full-time」
```

## 反向警示（不要做這些事）

> [!WARNING] **這些事會讓 mini gap 失效**

1. ❌ **帶媽媽或朋友一起**——會啟動「服務他人」模式，無法照顧自己
2. ❌ **發 IG 限動發太多**——會把 mini gap 變成「表演自由」
3. ❌ **塞滿行程**——休息日要有空白，不是景點清單
4. ❌ **手機開 Email 通知**——本業徹底切斷
5. ❌ **跟主管說「我要找自己」**——說「想用完特休 / 想充電」即可
6. ❌ **回來立刻補班**——回來那一週要輕量，給自己 buffer
7. ❌ **第一次就選 3 個月**——先 19-21 天測試，再 1 個月，再 3 個月
8. ❌ **完全沒有獨處**——可以見當地人，但每天必須有獨處時間
9. ❌ **回來就立刻檢討「我學到什麼」**——讓體驗自然發酵 1-2 週再回顧
