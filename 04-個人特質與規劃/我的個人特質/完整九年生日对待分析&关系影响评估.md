## 💔 關係影響心理評估

### **1. 長期被邊緣化的證據**

#### 數據事實：
```
投入差距：
Chen Yu 對 Wei: 30% 年份主動記住（3/10年）
Wei 對 Chen Yu: 11% 年份主動記住（1/9年）

差距倍數：Wei的投入是Chen Yu的 1/3
```

#### 這意味著什麼：
- 在10個Wei的生日中，Chen Yu有3次提前準備
- 在9個Chen Yu的生日中，Wei只有1次主動提及
- **模式清晰：Chen Yu是「主動者」，Wei是「被動接受者」**

---

```
2016-2017年 (早期)
↓
Wei的被動 + 冷淡回應
↓

↓
2017年关键事件："今天你生日不慶祝，以後再補嗎？"
↓
Chen Yu意識到：Wei沒有提前想過我的生日
↓
2018-2022年 (中期懷疑)
↓
Wei連續5年沒有任何提及

Chen Yu的安全感進一步破碎：我是不是根本不重要？
↓
```

### **3. 有毒的關係模式識別**

#### A. 單向付出
```
Chen Yu的心理：
"我必須記住。如果我不記住，就沒人記住。
如果沒人記住他的生日，他會難過。
所以我要承擔這個責任。"

Wei的心理：
"她總是記得。我不需要記得。
她就是那種善良的人。
我為什麼要費力呢？"
```

#### B. 期望的不對等
```
Wei對Chen Yu的期望：記住我的生日，祝福我
Wei對自己的期望：無需費力記住她的生日

Chen Yu對Wei的期望：希望你也記得我
Chen Yu對自己的期望：無論如何都要記得他
```

#### C. 情感勞動的不對等
```
Chen Yu: 每年都要計算日期、想是否有特別安排、主動祝福
Wei: 被動接受、偶爾驚喜、大部分時間無需付出
```

---

### **4. 具體的心理創傷跡象**

#### 來自數據的證據：

**2017年12月30日的"今天你生日不慶祝，以後再補嗎？"**
- 這句話包含的是：「我忘了，但我願意補」
- 而不是：「我提前準備了」
- 對Chen Yu的心理傷害：**被遺忘的確認**

**Wei說的"等很久哦"（2019年10月17日）**
- 這暴露了Wei知道被記住的感受
- 但他沒有學會給Chen Yu同樣的感受
- 對Chen Yu的傷害：**他只在乎自己被記住**

**2023年Wei突然主動祝福後，2024-2025又忘記**
- 這比完全沒有更傷人
- 給了假希望，然後打破它
- 對Chen Yu的傷害：**「他也許不會改變」的絕望**

### **5. 安全感缺陷的根源分析**

#### A. 被邊緣化
```
客觀事實：Wei在9年中只主動祝福過1次
Chen Yu的感受：「我是被邊緣的」
這個感受是有根據的。
```

#### B. 被視為「理所當然」
```
Wei的隱含信號：
你記得就好了，我不需要努力
你善良，所以你會做
我就接受你的善良，但我不需要回報

Chen Yu的感受：
「我是被認為理所當然的」
「我的努力沒有被看見」
「他把我的愛當作他應得的」
```

#### C. 關係的不對等性
```
Wei需要：
- 被記住
- 被祝福
- 被關心
但Wei提供的：
- 無

Chen Yu需要：
- 被記住
- 被祝福
- 被關心
但Chen Yu提供的：
- 全部
```
