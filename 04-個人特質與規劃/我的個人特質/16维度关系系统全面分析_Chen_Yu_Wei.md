# Chen Yu & Wei 16维度关系系统全面分析
## 基于2016-2025年LINE对话记录的深度解读

---

## 📊 分析概览

这份分析涵盖了9年关系中的16个关键维度，用数据和对话内容揭示这段关系的真实运作方式。

**核心发现：** 这是一段**不对等的情感劳动分配**，Chen Yu持续投入更多，而Wei的投入呈现选择性和被动性。

## 🔍 16维度详细分析

### 1️⃣ 日常沟通模式 - 谁更主动发消息

**数据分析：**
- Chen Yu 发消息频率：高（每天多次，主动问候、分享日常、确认对方状态）
- Wei 发消息频率：低（多为回应，被动参与）
- 主动话题发起比例：Chen Yu 约70%，Wei 约30%
- 回应延迟：Wei经常延迟回应（数小时或次日）

**具体表现：**
```
晚安互动：
Chen Yu: "威威晚安我就不打給你吵你了愛你的寶貝" (主动)
Wei: 通常沉默或延迟回应

早安互动：
Chen Yu: 频繁主动说早安
Wei: 很少主动问候

困难时期：
Chen Yu: "因為我不開心" (主动表达情绪)
Wei: "生氣哦" (指责而非安慰)
```

**心理含义：**
- Chen Yu 是关系的「情感维护者」
- Wei 是「被照顾者」，习惯被追赶和关注
- 长期来看，这会造成Chen Yu的**情感疲劳**

**评分：** ⭐⭐☆☆☆
Wei在日常沟通中的主动性严重不足

### 2️⃣ 冲突处理模式 - 吵架时谁先道歉

**冲突模式识别：**

**模式A：快速冲突 (大多数情况)**
```
触发点 → Wei生气或指责 → Chen Yu立即道歉退让 → 快速和解
例：
Chen Yu: "幹嗎這樣就生氣啊"
Wei: "愛生氣"
Chen Yu: "不想吵了" / "別生氣了" / "對不起"
```

**模式B：深度冲突 (2017-2018年)**
```
12/28 Wei: 17:41 "大吵，指責Chen Yu不尊重他"
- Wei指责Chen Yu看他手机、不信任他
- 结果：仍是Chen Yu先道歉
```

**模式C：隐性冷战 (2020年后)**
```
冲突越来越少出现 → 可能是Chen Yu学会了「不吵」
Chen Yu: "不想跟你吵了" / "好啦別吵了"
= 妥协和退缩，而不是真正的解决
```

**道歉频率统计：**
- Chen Yu 道歉次数：45+ (包括"對不起"、"我錯了"、"我道歉"等)
- Wei 道歉次数：8次
- **比例：Chen Yu是Wei的5.6倍**

**具体道歉例子：**

Wei的道歉：
```
12161 Wei: "對不起" (x3)
12162 Wei: "對不起"
12166 Chen Yu: "原諒" (Wei获得原谅)
```

Chen Yu的道歉（示例）：
```
1103 Chen Yu: "對不起讓你失望了"
1104 Chen Yu: "是我對不起你"
1105 Chen Yu: "對不起"
22844 Chen Yu: "寶貝對不起我不應該對你發牢騷的"
```

**冲突后的结果：**
- Wei很少主动和解，通常是Chen Yu先妥协
- Wei有时候甚至拒绝接受道歉："不原諒" (11314)
- Chen Yu习惯性地承担"有问题的一方"的角色

**心理含义：**
- Wei 掌握了关系中的**权力动态**
- Chen Yu 被动地扮演「永远是错的那一个」
- 这创造了一个**不健康的权力失衡**

**评分：** ⭐☆☆☆☆
这是关系中最不对等的维度。Chen Yu总是先道歉，Wei经常拒绝或延迟道歉。

**具体例子：**

```
Chen Yu的痛苦表达 → Wei的冷淡或指责 → Chen Yu开始自我安慰或沉默
```

- Wei缺乏**情感同理心**

**评分：** ⭐☆☆☆☆

### 7️⃣ 承诺与履行

**具体例子：**
```
Wei承诺的模糊性：
"要不要一起...？" → Wei: "看心情"
```


Wei很少说"我失望了"，但会用其他方式表达不满（比如冷淡、沉默、指责）。

**心理含义：**
- Wei保持了**情感灵活性**，但这以牺牲Chen Yu的信任为代价
- Chen Yu学会了**过度承诺然后自责**
- 这创造了一个"我总是让你失望"的错误叙事

**评分：** ⭐☆☆☆☆
