## 十三、關係模式深層更新（2026-04-06 新增）

### 13.0 核心發現：自信缺乏是所有關係問題的根源

今天在探索「容易心動」這個現象時，挖到了更底層的東西：

```
表面問題：容易對別人動心、感情不穩定
↓
中層原因：掌控行為、焦慮型依戀
↓
底層根源：對自己沒有自信，需要從外界獲得「我夠好」的感覺
↓
最根源：童年情緒從來沒有被穩定接住——那面鏡子從一開始就是破的
```

**關鍵理解**：
- 不是「這個人特別吸引我」
- 而是「這個人給了我『我值得被愛、我夠好、我有吸引力』的感覺」
- 那個感覺對我來說是**缺貨很久的東西**，所以特別強烈
- 一旦伴侶習慣化、不再給這個感覺，我的神經系統就去尋找新的來源

### 13.1 五年前的出軌事件

```
事實：
- 五年前曾出軌
- 我對此幾乎沒有記憶（我選擇了遺忘）
- 男友 Wei 記得清楚，他稱之為創傷
- 他說他需要一個交代

心理機制（動機性遺忘 / Motivated Forgetting）：
- 這件事與我的自我形象嚴重衝突
- 帶來難以承受的羞恥感
- 大腦主動將記憶「封存」以保護自我
- 這不是有意撒謊，是神經系統的保護機制
- 但它讓 Wei 感到被二次傷害（「你連這件事都不記得？」）

當時的底層需求（現在的理解）：
- 那個人給了我「被看見、有吸引力、我夠好」的感覺
- 不是愛那個人，是愛「被他看見的自己」
- 伴侶關係習慣化後，那個「我夠好」的燃料斷了
- 新的人重新給了那個感覺，我的判斷能力暫時失效
```

**給治療師的討論點**：
- 動機性遺忘在我身上有多普遍？是否還有其他事件也被這樣封存了？
- 如何在不強迫記憶的情況下，讓 Wei 感到被認真對待？
- 我要如何向 Wei 給出一個「誠實的交代」，即使那個交代是「我現在才開始理解當時為什麼」？

### 13.2 容易心動的完整機制

```
觸發條件：
「有人給我一種『我很好、我被看見、我有吸引力』的感覺」

為什麼這個感覺特別強烈？
→ 原生家庭從來沒有穩定給過這個感覺

→ 問題是：從別人那裡借來的自信，是不穩定的
→ 一旦那個人不再給，就會再去找下一個來源

真正需要的：
→ 建立來自內部的自信——不依賴外界鏡射
→ 這是目前最核心、最缺乏的能力
```

**給治療師的討論點**：
