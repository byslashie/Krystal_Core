## 一、簡要問題描述

我意識到自己在親密關係中有**嚴重的掌控欲和焦慮**。我一直用教導和掌控來管理伴侶，當他不符合我的期望時會極度焦慮。現在他變得冷漠疏遠，我意識到這可能是我的掌控造成的。

## 二、主要症狀

### 焦慮症狀
```
✓ 當伴侶不告訴我見面時間時，我會焦慮
✓ 當他不聽我的建議時，我會焦慮
✓ 當他有獨立決定時，我無法接納
✓ 我害怕失去他，所以試圖掌控他
✓ 無法接受「無法預測」的事情
```

### 掌控行為
```
✓ 期望他按照我的計劃活（「我想要的是一個跟我有計畫的人」）
✓ 用長句子邏輯去說服他改變想法
✓ 當他做決定不符合我的期望時，我會說「我早說」
✓ 我一直是主動打電話的人（他後來停止主動）
✓ 我用「關心」的名義去控制他的決定
```

### 認知扭曲
```
✓ 「他如果不聽我的就是不愛我」
✓ 「我需要掌控才能感到安全」
✓ 「他應該主動、應該聽話」
✓ 「如果他不符合我的期望，我就無法愛他」
```

### 關係模式
```
✓ 我愛的是「符合我期望的他」，不是「真實的他」
✓ 當他反抗掌控時，我變得更焦慮和教導
✓ 他從主動、粘膩變成冷漠疏遠
✓ 我現在累了，不想再努力，甚至如釋重負他會離開
```

## 六、可能的診斷線索

根據我的自我觀察，我可能有：

```
□ 焦慮型依恋（Anxious Attachment）
  特徵：害怕被拋棄、過度需要確認、用掌控來緩解焦慮

□ 掌控欲強/完美主義（High Need for Control/Perfectionism）
  特徵：期望他人符合標準、無法接納不同、用邏輯壓倒他人

□ 關係共依存（Codependency）
  特徵：用他人的反應定義自我價值、過度投入、忽視自己需求

□ 焦慮障礙的特徵（Anxiety Disorder Features）
  特徵：分離焦慮、廣泛焦慮、強迫性思考

□ 強迫型人格特徵（Obsessive-Compulsive Traits）
  特徵：過度計劃、需要控制、難以放手
```

## 十一、2026年3月16日深層探索的突破

今天完成冰山潛行工作表並進行深度自我探索後，有以下重要發現：

### 核心發現1：最深層恐懼——「不被回應」
```
根源：童年哭泣時沒有人來，情緒長期不被接住
媽媽傳遞：哭=軟弱、哭=不好的情緒
媽媽行為：至今講電話還會直接掛掉，不說掰掰

內化結果：我的感受、我的存在，可以被單方面切斷
長大觸發：Wei 不回訊息 → 觸發「又要被切斷了」的深層恐懼
行為反應：瘋狂看訊息、秒讀秒回要求、緊抓不放
```

### 核心發現2：堅強外表是保護機制
```
形成原因：哭泣換來的是懲罰或忽視，所以學會把情緒壓下去
外表呈現：能幹、獨立、掌控全局、邏輯強
保護功能：真實有效地保護了自己走到今天
代價：擋住了傷害，也擋住了真正的親密連結
今天突破：第一次允許自己大哭崩潰——那道牆開始裂開了
```

### 核心發現3：焦慮與控制欲的關係
```
焦慮是根，控制欲是花，它們是同一件事。

童年不安全環境 → 神經系統設定為高度警戒
↓
任何不可預測的事 → 焦慮警報觸發
（字寫不好/考試沒考好/犯錯 → 危險）
↓
長大後警報系統仍在運作，但觸發點換了
（Wei不回訊息/不聽話/事情失控 → 危險）
↓
大腦的解法：控制它，焦慮就消失
↓
控制欲出現
↓
對方反抗 → 更焦慮 → 更想控制
↓
惡性循環
```

### 核心發現4：從未被無條件接受過
```
童年收到的愛是：「你做對了才值得被接受，做錯了有後果」
這不是無條件的愛，是「表現換取接受」

形成的核心信念：
→ 「我不夠好」
→ 「我的存在是有條件的」
→ 「犯錯是危險的」
→ 「沒人需要我」
→ 「我必須夠好，才值得被愛」

這個信念進入了每一段親密關係。
```

### 核心發現5：代際傳遞
```
阿嬤重男輕女
↓
媽媽在「不被完全接受」的環境長大
↓
媽媽用高標準、掌控來確保孩子「夠好」
↓
我繼承掌控模式，姐姐發展出「必須向外證明自己很棒」
↓
我在親密關係中複製媽媽的掌控，Wei 逐漸變得像爸爸一樣冷漠
↓
我最害怕的事，被自己的行為創造出來了

這是跨代的傷，不是我的錯，但改變是我的責任。
```
