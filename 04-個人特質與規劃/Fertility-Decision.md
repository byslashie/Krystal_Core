## 為什麼凍 = 把選擇權買回來

### 不凍的真實後果

```
30 歲（現在）
  ↓
焦慮：「再不確定關係就來不及生」
  ↓
選項 A：將就現有關係（為了不錯過生育期）
選項 B：倉促離開找新人（為了重新開始）
  ↓
無論 A 或 B 都不是「自由的選擇」
  ↓
35 歲後生育成本飆升（試管比例急遽上升）
```

### 凍的真實意義

```
30-31 歲：凍卵（取 15-25 顆 30 歲時的卵）
  ↓
35-42 歲：想用就用，不想用也 OK
  ↓
找伴侶 = 找對的人，不是「來得及的人」
人生決定 = 從容判斷，不是時間倒數
  ↓
焦慮真正解除：「我有時間了」
```

**凍卵 = 給未來 5-10 年的自己一份禮物**

## 凍卵的真實數據（2026 台灣行情）

### 成功率（最關鍵指標）

| 取卵年齡 | 每顆解凍受孕率 | 達成 1 胎需要的卵數 |
|---|---|---|
| **30-32 歲** ⭐ | ~7% | **15-20 顆** |
| 33-35 歲 | ~5% | 20-30 顆 |
| 36-37 歲 | ~3% | 30-40 顆 |
| 38+ 歲 | <2% | 50+ 顆 |

> **30 歲的你 = 最划算的窗口期**。Krystal 你現在就在金窗口入口。

### 金錢成本

| 項目 | 金額（NT$） |
|---|---|
| 單次療程（藥物 + 取卵 + 第一年保存） | 12-18 萬 |
| 一次取卵數 | 10-15 顆（30 歲） |
| 建議療程數（為達 15-20 顆） | 2-3 次 |
| **總成本（一次性）** | **30-50 萬** |
| 每年保存費 | 6,000-12,000 元 |
| 預期保存 8-10 年 | 累計 5-12 萬 |
| **10 年總成本估算** | **35-60 萬** |

### 身體成本

- 打排卵針：10-12 天（自己肚皮注射）
- 取卵手術：全麻 15 分鐘
- 恢復期：1-2 週
- 副作用：
  - 常見：腹脹、情緒波動、暫時體重增加
  - 罕見（<1%）：卵巢過度刺激症候群（OHSS）

### 時間成本（從諮詢到取卵）

```
第 1 個月：諮詢診所（建議比較 2-3 家）
第 1-2 個月：驗血、超音波、評估卵巢儲備（AMH）
第 2-3 個月：等待適合的月經週期啟動
第 3 個月：打針 10-12 天 → 取卵 → 恢復
第 4 個月：評估卵數 → 必要時規劃第二次
```

**從決定到完成第一次取卵：4-5 個月**

## 診所評估準則

挑診所看 5 件事：

### 1. 取卵成功率
- 看「**每顆成熟卵的解凍受孕率**」（不是「成功懷孕案例數」這種行銷數字）
- 要求看 30-32 歲族群的真實統計

### 2. 醫師專業度
- 主治醫師年資（≥ 10 年較穩）
- 是否專精 IVF / 凍卵（而非一般婦產科兼任）

### 3. 實驗室設備
- 玻璃化冷凍技術（vitrification）= 現代標準
- 培養箱品質（time-lapse 較好）

### 4. 費用透明度
- 是否包含所有藥物（有些診所單獨收費）
- 第二次療程是否有優惠
- 保存費漲價機制

### 5. 服務體驗
- 諮詢時的回答清晰度
- 護理人員態度
- 預約彈性

## 心理準備清單

### 凍卵前要面對的事

- [ ] 「**為什麼要凍**」想清楚（不是因為焦慮、是因為自由）
- [ ] 「**如果未來沒用，會後悔嗎**」（80% 凍的人沒用、但不後悔）
- [ ] 「**家人 / 男友會反對嗎**」（先想好怎麼溝通）
- [ ] 「**療程期間的工作安排**」（嘉實請假規劃）
- [ ] 「**身體不適時的支援系統**」（誰陪你回診、誰知道）

### 療程中的心理現象（常見）

- 打針期：情緒起伏大（賀爾蒙影響）
- 取卵當天：意外的「**這就結束了**」空虛感
- 取卵後 1 個月：強烈反思「**我到底想要什麼**」
- 6 個月後：許多人說「**早知道早點做**」
