# Threads 文體模板

## 故事型（個人覺察）

```
我最近才發現...
[核心想法，1-2句]

這個想法來自 [背景或事件]

現在我懂了，[轉折點或學到的東西]

你呢？你有類似的經驗嗎？
```

### 範例
```
我最近才發現，高共感女性的最大問題，不是「太敏感」，而是「把別人的情緒當成了自己的責任」。

這個想法來自我諮商時的一個頓悟——我用十年的人生去防備一個人會不會傷心，結果忘記了問自己開不開心。

現在我懂了，設定邊界不是冷漠，而是尊重兩個人都有權利管理自己的情緒。

你呢？你是不是也習慣了「先照顧別人，再來照顧自己」？
```

## 價值型（可操作的洞察）

```
給高共感的你：
[問題陳述]

我試過的解決方案：
1. [方法 A]
2. [方法 B]
3. [方法 C]

最有效的是 [哪個方法]，因為 [原因]

如果你也在為 [問題] 苦惱，下次可以試試。
```

### 範例
```
給高共感的你：你是不是習慣在被批評時，第一反應是「對不起，我改」，而不是「這個建議對嗎」？

我試過的解決方案：
1. 當場接受所有批評（結果越來越沒自信）
2. 全部駁回（得罪很多人）
3. 暫停，深呼吸，問自己「這個批評對我有幫助嗎」

最有效的是第 3 個，因為它讓我有時間分辨「這是我的問題」還是「這是別人的投射」。

如果你也在為「不知道該不該相信自己的判斷」苦惱，下次可以試試。
```

## 提問型（引發互動）

```
[觀察或問句]

我的感覺是 [個人觀點]

但我也知道有人會說 [對立觀點]

你怎麼想？[關鍵問題]
```

### 範例
```
有沒有人發現，「要獨立自主」和「允許自己依靠別人」，這兩件事其實不衝突？

我的感覺是，真正的獨立，是「知道自己何時需要幫助，也知道何時要放手」。

但我也知道有人會說，「依賴別人就會喪失掌控感」。

你怎麼想？你覺得獨立和相信別人可以共存嗎？
```

## 交易/系統型

```
[簡單敘述一個交易或系統洞察]

關鍵是 [核心邏輯]

如果你 [具體行動]，可能會發現 [結果]

有人試過嗎？
```

### 範例
```
有個有趣的觀察：我發現大部分人買股票時都是高估能力，停損時都是低估時間。

關鍵是「認知偏差」——我們高估短期判斷力，低估長期複利力。

如果你把「我能預測股價」改成「我相信系統和時間」，可能會發現交易反而變簡單了。

有人試過嗎？
```

## 反思型（週末常用）

```
這週 [發生了什麼]

我想到 [反思的東西]

下週我想試試 [具體改變]

希望 [期待的結果]
```

### 範例
```
這週連續虧損了三次，心情很差。

我想到，如果我更信任系統而不是情緒，可能就不會在最關鍵時刻改變決策。

下週我想試試「決策後就靜音市場，直到週五才看一次」。

希望這次能真的讓我體驗「紀律的力量」。
```

## ❌ 避免的寫法

### 太長
```
❌ 我最近在思考一個非常複雜的人生問題，
就是如何在保持自我獨立性的同時，
也能夠維持一段親密的人際關係...
```

### 太短（沒內容）
```
❌ 今天天氣很好
```

### 太教訓人
```
❌ 你們都應該像我一樣，認識自己、設定邊界、成為獨立女性
```

### 過度包裝（不真實）
```
❌ 我完全解決了我所有的心理問題，現在每一天都閃閃發光！
```

---

## ✅ 好的 Threads 特徵

- **有停頓感**：不是一大段，而是用分行和短句
- **誠實**：願意承認還沒解決、還在掙扎
- **有點感**：讓人覺得「我也是這樣」或「我沒想過這個角度」
- **邀請對話**：以問句結尾，留空間給讀者
- **具體**：有例子或數字，不是空泛的概念
