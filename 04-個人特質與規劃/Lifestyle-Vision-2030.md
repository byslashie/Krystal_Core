# Lifestyle Vision 2030 — 五星級生活 × 地理自由

> [!QUOTE]
> 我不需要永遠住在某個城市，我需要的是隨時能去任何城市的能力。
> 系統替我賺錢，世界當我家。

---

## 真實起點（Reality Check, 2026-05-13）

> [!IMPORTANT] **目前是朝九晚五上班族，不是 freelancer。**
> 不要被 vault 裡的「09-12 交易、13-17 PM」時程誤導——那是**願景中的日程**，不是現況。
> 規劃必須建立在「**從上班族 → lifestyle**」的轉換路徑上，不是「假設已經自由」。

| 項目    | 現況                          |
| ----- | --------------------------- |
| 本業    | 嘉實資訊 量化交易 PM（朝九晚五、一週 1 天遠端） |
| 對本業感受 | A — 喜歡，只是想偶爾喘息（**不需要逃離**）   |
| 副業    | 交易 + 0050 定期定額 1.8 萬/月      |
| 資產    | 500 萬（2026-05）              |
| 居住    | 住家裡，居住成本 0                  |
| 特休    | 15 天/年，可全修；2026 剩餘 5 天      |
| 留停制度  | ✅ 公司有                       |
| 內在動機  | 「翻身感」（用「選擇權」語言重新定義中）        |
| 最怕的劇本 | 達自由但失健康/關係（C > B > A）       |

**關鍵優勢**：本業 = 量化交易，副業也是量化交易 → **知識直接複利，不需要換跑道**。

## 核心畫面（One-Sentence Vision）

> **在自己的房子裡像住五星級飯店一樣醒來，一年有 3-6 個月，世界其他角落也是我的家。**

---

## 願景五要素

### 1. 居住空間（Home as a Sanctuary）

**地點**
- 台灣自有房（**不限台北**，優先考慮：高雄、台南、宜蘭、台中精華區）
- 過渡期（~2028 前）：**住家裡，居住成本 0**，全力把錢投入資產
- 永久基地：35-45 坪，山景/海景/綠意至少擇一

**空間質感（五星級飯店感）**
- 浴室：
  - 獨立浴缸（深泡、可看景）
  - **橢圓形雙面盆**（自己 + 伴侶各一）
  - 乾濕分離 + 大尺寸淋浴間
  - 暖氣 / 暖風機 / 浴巾加熱架
- 臥室：
  - King size 床 + 五星級寢具（埃及棉、羽絨）
  - 遮光窗簾 + 床頭可調燈
  - 衣帽間（walk-in closet）
- 客廳 / 書房：
  - 大面採光 + 落地窗
  - 一張能讓我安心交易 4 小時的工作桌
  - 質感沙發 + 閱讀角
- 廚房：
  - 中島 + 開放式
  - 高蛋白備餐空間

### 2. 地理自由（3-6 個月海外旅居）

**模式**：深度旅居，非觀光
- 每次至少停留 1 個月，融入當地生活
- 一年 2-4 個 base，不追求多國輪替

**候選 base 城市**
| 城市 | 月支出（中等水準） | 適合季節 | 主題 |
|---|---|---|---|
| 京都 / 東京 | 12-18 萬 | 4-5 月、10-11 月 | 美感、寧靜、櫻花/楓葉 |
| 清邁 | 5-8 萬 | 11-2 月（涼季） | 養生、社群、低成本 |
| 里斯本 / 波多 | 10-15 萬 | 5-9 月 | 歐洲基地、自媒體圈 |
| 峇里島烏布 | 6-10 萬 | 4-10 月 | 瑜伽、冥想、療癒 |
| 首爾 | 12-16 萬 | 春秋 | 美食、購物、文化 |

**旅居方式**
- Airbnb 月租型公寓 / 服務式公寓
- 帶得走的工作（交易、寫作、線上會議）
- 至少一個 base 有「我喜歡的健身房 + 咖啡廳 + 市場」三件套

### 3. 收入結構（可攜帶 × 半被動）

**核心原則**：不被地點綁、不被時間綁、可預期

| 收入來源 | 2026 現況占比 | 2030 目標占比 | 地理自由度 |
|---|---|---|---|
| **嘉實本業薪水** | **~70%（主力）** | 0%（已脫離） | ⭐ |
| 系統化交易（量化 + 波段） | 20% | 50% | ⭐⭐⭐⭐⭐ |
| ETF / 股息 / 利息（含 0050） | 10% | 15% | ⭐⭐⭐⭐⭐ |
| 委託資金管理分潤 | 0% | 20% | ⭐⭐⭐⭐ |
| 抒發型自媒體（副產品） | 0% | 10% | ⭐⭐⭐⭐⭐ |
| 其他（合作、版稅） | 0% | 5% | ⭐⭐⭐⭐ |

**不做的事**
- ❌ 需要每天打卡的工作（**長期目標**，2028 後考慮脫離）
- ❌ 需要固定地點才能執行的服務業
- ❌ 線性換時間的接案（除非單價極高且短期）
- ❌ **商業型自媒體**（KPI 化、頻率化、漲粉化）—— 會吃掉假日

### 4. 日常節奏（Lifestyle Rhythm）

**台灣 base 日（一週 5-6 天）**
```
07:00  自然醒（不設鬧鐘）
07:30  浴缸 + 早餐 + 市場簡報（看 Discord bot 推送）
09:00  交易系統檢查（30 分鐘以內，靠自動化）
10:00  重訓 / 瑜伽 / 散步
12:00  午餐 + 小休
14:00  深度工作（策略研究 / 內容創作 / 學習）
17:00  輕量交易回顧
18:00  晚餐 + 伴侶時間 / 朋友
21:00  閱讀 / 冥想 / 日記
22:30  睡眠
```

**海外 base 日（旅居期）**
- 早上：在地咖啡廳 + 簡單交易檢查
- 中午：探索當地、運動、學語言
- 下午：深度創作（在不同環境寫作很有靈感）
- 晚上：當地朋友、文化體驗

**目標總工時**：一週「必須親自做」的工作 ≤ 15-20 小時（不是 4 小時，誠實一點）

### 5. 關係與內在（Relationships & Inner World）

**伴侶**（對齊 Vision Board 已寫的條件）
- 能一起旅居（不只是觀光）
- 財務獨立，不互相綁
- 共享健康 / 閱讀 / 成長節奏

**社群**
- 台灣：3-5 個能深度對話的朋友
- 海外：每個 base 有 1-2 個當地連結
- 線上：交易 / 創作 / 量化的同溫層

**內在**
- 不被「應該」綁架（不必為了 KOL 而 KOL、不必為了財務自由而過勞）
- 持續諮商 / 冥想 / 身體覺察
- 每年 1 次完全斷網 7 天

## 財務試算（基於「居住成本 0」前提）

### 月支出估算（過渡期，住家裡）

| 項目 | 金額 |
|---|---|
| 基本生活（吃、交通、保險、娛樂） | 3-4 萬 |
| 健康（健身房、營養品、諮商） | 1-1.5 萬 |
| 學習 / 訂閱 / 工具 | 0.5-1 萬 |
| 旅遊（攤平，2-3 次/年） | 1.5-2.5 萬 |
| 緩衝 | 1 萬 |
| **合計** | **7-10 萬 / 月** |

> 對齊 Vision Board 寫的「每月支出 ≤ 2.5 萬」——那是極度節省版，
> 這份是「五星級 lifestyle 過渡期」的真實數字。
> 居住成本 0 讓總額仍比租精華區低 50% 以上。

### 月支出估算（永久版，自有房 + 海外旅居 4 個月/年）

| 項目 | 金額 |
|---|---|
| 自有房維護（管理費、水電、稅、保養） | 1.5-2.5 萬 |
| 基本生活 | 4-5 萬 |
| 健康 / 學習 | 1.5-2 萬 |
| 海外旅居攤平（4 個月 × 10 萬） | 3.5 萬 |
| 緩衝 + 彈性 | 1.5 萬 |
| **合計** | **12-15 萬 / 月**（年 150-180 萬） |

### 對應財務自由門檻（4% rule）

| 月支出             | 需要資產            | 預估達成年     |
| --------------- | --------------- | --------- |
| 7-10 萬（過渡期）     | **2100-3000 萬** | 2027-2028 |
| 12-15 萬（永久版）    | **3600-4500 萬** | 2028-2029 |
| 含買房 1500-2500 萬 | **5000-7000 萬** | 2030-2031 |

> 對齊 Long-Term-Plans：2029 達 5000 萬 ≈ 永久版 lifestyle 啟動點 ✅

## 雙軌路徑（A 自由優先 / B 1 億 Upside）

> [!IMPORTANT] **不二選一，並行追蹤。**
> A 軌是「**保底自由**」（自己過得爽）—— 主路徑、必達。
> B 軌是「**順其自然 upside**」（影響力 + 餘裕）—— 達到 A 後若還有動力與興趣，自然往 B 走。
>
> 不再用「翻身」當動機。改用「**選擇權**」：
> - 3000 萬 = 給未來自己「想不想做就不做」的選擇權
> - 1 億 = 給未來自己「想做大事業就做」的選擇權

### A 軌（保底自由）：3000-3500 萬 → Lifestyle 啟動

| 年 | 資產目標 | 對應 lifestyle 階段 |
|---|---|---|
| 2026 | 650 萬 | 上班族 + 副業累積 |
| 2027 | 830 萬 | 首次真正 mini gap（15 天獨旅） |
| 2028 | 1060 萬 | 留職停薪 1-3 個月試水 |
| 2029 | 1346 萬 | 評估是否辭職 / 轉兼職 |
| 2030 | 1675 萬 | 半 gap lifestyle 啟動 |
| 2033 | **3341 萬** | **完整 lifestyle 啟動點** ⭐ |

> 假設：中性 25% 年化報酬 + 月投 1.8 萬持續到 2030（之後若辭職則停止月投）

### B 軌（順其自然 upside）：1 億 → 影響力 + 餘裕

| 年 | 資產目標 | 備註 |
|---|---|---|
| 2035 | 5247 萬 | 中性情境自然成長 |
| 2037-2038 | **~1 億** | **沒有死線、達不到也 OK** |

> [!NOTE] **1 億不再是「2033 死線」，而是「複利數學的自然結果」。**
> 達到 A 軌 3000 萬後，你可以選擇：
> - **停下來**（這就是 lifestyle，不必繼續衝）
> - **繼續複利**（不主動加碼，讓資產自然滾動到 1 億）
> - **加碼影響力**（如果這時自媒體/委託管理有突破，往 B 軌衝）
>
> 這個決定**現在不用做**。**3000 萬時的你**比現在的你更清楚自己想要什麼。

## 分階段路徑（與 Long-Term-Plans + Mini-Gap-Launch-Plan 對齊）

### 階段 0：上班族 × 累積期（2026，現在）
- **本業**：嘉實量化交易 PM，正常上班
- **副業**：交易系統 + 0050 月投 1.8 萬
- **特休**：2026 剩 5 天，**保留作彈性 + 12 月可能 5-7 天 lifestyle 預演**
- **居住**：家裡（成本 0）
- **里程碑**：2026 末資產 ~650 萬

### 階段 1：首次 Mini Gap（2027 Q1）
- **本業**：照常上班
- **動作**：用新一年的 15 天特休 + 週末，做 19-21 天**獨旅**
- **核心要求**：**一個人**，不帶媽媽、不帶朋友 ⭐
- **目的**：第一次真正照顧自己、測試 lifestyle 預覽
- **里程碑**：2027 末資產 ~830 萬

### 階段 2：留職停薪試水（2027 Q4 - 2028）
- **本業**：留職停薪 1-3 個月
- **動作**：選一個喜歡的 base 深度住
- **目的**：測試「沒有正職的真實感覺」+ 副業能否撐起生活
- **里程碑**：2028 末資產 ~1060 萬

### 階段 3：選擇點（2029-2030）
- 已體驗過 mini gap + 留停
- 此時做真正的決定：
  - **路徑 A1**：回嘉實繼續上班 + 一年 2-3 次 mini gap（穩定派）
  - **路徑 A2**：辭職全力做交易 + 抒發創作（自由派）
  - **路徑 A3**：轉兼職 / 顧問（中間派）
- **里程碑**：2029 末資產 ~1346 萬

### 階段 4：尋找永久基地（2030-2032）
- **居住**：
  - 研究台灣非北部買房選項
  - 出手買房（高雄/台南/宜蘭/台中），約 1500-2500 萬
  - 設計浴缸 + 橢圓雙面盆 + 工作空間
- **生活**：一年國外旅居 3-4 個月（不同 base 試）
- **里程碑**：2030 末資產 ~1675 萬

### 階段 5：完整 Lifestyle（2033+）
- **居住**：自有房 = 五星級避世空間
- **生活**：一年 3-6 個月海外，找出最愛的 2-3 個 base
- **收入**：被動 + 半被動覆蓋全部支出
- **里程碑**：2033 資產 3000-3500 萬（A 軌達成 ⭐）
- B 軌持續累積，**無死線**

## 抒發型自媒體原則（重要）

> [!IMPORTANT] **自媒體採「抒發型」路線，不是「商業型 KOL」路線。**
> 你的擔心是對的——「**多經營自媒體 = 假日時間減少**」是商業型的宿命。
> 抒發型不會吃掉假日，**它本身就是假日**。

### 兩種模式的差別

| | 商業型自媒體 | **抒發型自媒體（你的選擇）** |
|---|---|---|
| 目標 | 漲粉、變現、KOL | 自我整理、療癒、紀錄 |
| 頻率 | 規律高頻（每週 N 篇） | **想寫才寫** |
| 內容 | 投讀者所好 | **投自己所需** |
| 時間 | 吃掉假日 ❌ | **就是假日本身** ✅ |
| 收入 | 主動經營來的 | **副產品**（粉絲被真實感吸引） |
| 持久度 | 容易燃燒殆盡 | **可寫一輩子** |

### 抒發型 6 原則

1. **不設發文頻率**——想寫才寫，可能一週三篇，可能一個月一篇
2. **不追數字**——不看互動率，不焦慮粉絲數
3. **長文 > 短影音**——文字是抒發，影音是表演（消耗更大）
4. **單一平台聚焦**——選一個你最不抗拒的（Threads？個人 blog？Substack？）
5. **內容是「整理已經發生的事」**——不是「為了發而做新的事」
6. **變現是副產品**——可能 1-2 年後才有，可能沒有，**都 OK**

### 為什麼這對 Krystal 特別合適

- ✅ 你 Vision Board 寫「**真實生活紀錄者**」「**有靈魂的量化者**」——抒發型才能達成
- ✅ 你目前最怕「失健康/關係」（Q3 排序 C > B > A）——抒發型不過勞
- ✅ 你住家裡、有正職薪水——**不需要靠自媒體賺錢**
- ✅ Mini gap 期間的「**獨旅紀錄**」剛好是天然素材
- ✅ Tim Ferriss、Khe Hy 都是這樣開始的——**不是為了當 KOL 才寫**

### 啟動方式（不急）

不必為了「啟動」而啟動。建議順序：
1. **2026 內**：把心情、交易感受、學習筆記寫在 Obsidian（**私人**，不發）
2. **2027 第一次 mini gap 期間**：嘗試寫旅居日記，可選擇性公開 1-2 篇
3. **2027 後**：如果寫得開心，找一個平台慢慢公開
4. **2028+**：如果有人關注，自然會變成資產

> 這個順序確保：**你不會為了流量犧牲真實感**。
