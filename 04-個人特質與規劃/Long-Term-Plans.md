# 長期發展規劃（Long-Term Plans）

> [!IMPORTANT] **2026-05-13 重大校準**
> 原版以「2025 YOY +94.18%」為基準線推估「2026 達 1000 萬」，過於樂觀。
> 校準後使用「**中性 25% 年化 + 月投 1.8 萬**」的現實情境，並對齊真實前提：
> - **目前是嘉實量化 PM 上班族，不是全職交易者**

## 3年計畫（2026–2028）：上班族 × 副業累積 × Mini Gap 試水
*重心：保住嘉實本業、累積副業能量、用特休/留停測試 lifestyle，資產 500 → 1060 萬*

### 職業（Career）
- [ ] **2026 全年**：嘉實量化 PM 本業照常，**不急著轉全職交易**
- [ ] **2026 Q3**：完成 Phase 1 剩餘 50%（3 套核心策略 + D1/A1 初版上線）
- [ ] **2026 Q4**：完成 Phase 2 自動化系統（Python 回測 + 決策引擎串接）
- [ ] **2027 Q1**：**首次真正 mini gap**（15 天特休獨旅，詳見 [[Mini-Gap-Launch-Plan]]）
- [ ] **2027 Q4 / 2028**：**留職停薪 1-3 個月試水**
  - 觸發條件：副業現金流穩、跟主管溝通過、心理準備好
- [ ] **2028 末**：評估是否承接委託資金（不勉強，看 mini gap 後狀態）
- [ ] **持續**：建立個人品牌：[[量化作品集]] 上線（職業可信度錨點）

### 財務（Finance，中性 25% 情境）
- [ ] **2026**：500 → 650 萬（**改以 25% 為基準，不追 100%**）
- [ ] **2027**：650 → 830 萬
- [ ] **2028**：830 → 1060 萬
- [ ] 0050 月投 1.8 萬不間斷
- [ ] 持續優化策略系統：[[Krystal 專職交易者八字 × 策略系統白皮書 V1.0]]

### 體態與能量（Vitality）

- [ ] 長期穩定節奏：重訓 3x + 有氧 2x + 休息 1x

## 5年計畫（2029–2031）：選擇點 × 尋找永久基地
*重心：資產 1346 → 2112 萬，做出「上班 vs 自由」的真正選擇，研究買房*

### 職業（Career）
- [ ] **2029**：經歷過 mini gap + 留停後，做出選擇
  - **A1**：回嘉實繼續上班 + 一年 2-3 次 mini gap（穩定派）
  - **A2**：辭職全力做交易 + 抒發創作（自由派）
  - **A3**：轉兼職 / 顧問（中間派）
- [ ] **2029 末**：資產達 1346 萬
- [ ] **2030-2031**：根據選擇執行
  - 若 A2：開始承接小規模委託資金管理
  - 若 A1/A3：累積資產到 2000 萬+

### 居住（重要新增）
- [ ] **2030 起**：研究台灣**非北部**買房選項（高雄/台南/宜蘭/台中精華區）
- [ ] **2031 評估**：出手時機，目標 1500-2500 萬的房子
- [ ] 設計重點：浴缸 + 橢圓雙面盆 + 五星級空間質感（詳見 [[Lifestyle-Vision-2030]]）

### 伴侶與關係（Partner & Relationships）
- [ ] 共同規劃未來生活藍圖（財務規劃對齊、生活節奏共識）
- [ ] 深化情感連結，共同面對生命挑戰

### 生活方式
- [ ] **2030**：半 gap lifestyle 啟動，一年國外旅居 3-4 個月
- [ ] 投資組合多元化（台股 + 美股 + 另類資產）

## 10年計畫（2032–2035）：完整 Lifestyle × 上限版自由
*重心：3000-3500 萬 = A 軌 lifestyle 啟動點 ⭐ ／ 1 億為 B 軌 upside（無死線）*

### 職業（Career）
- [ ] **2033**：資產達 **3000-3500 萬**，**A 軌 lifestyle 啟動** ⭐
- [ ] 工作純粹「**因為喜歡**」，不為錢
- [ ] 若有興趣：建立抒發型自媒體 → 自然累積成資產
- [ ] B 軌：1 億不設死線，**達不到也 OK**

### 居住與 Lifestyle
- [ ] 自有房 = 五星級避世空間（橢圓雙面盆 ✅）
- [ ] 一年 3-6 個月海外旅居，找出最愛的 2-3 個 base
- [ ] 「**整全的人生 > 數字達標**」

### 伴侶與關係
- [ ] 擁有成熟、穩定、充滿安全感的長期伴侶關係
- [ ] 能一起旅居（不只是觀光）

### 願景與精神
- [ ] 回饋社會：不給答案，給框架——「**你也可以用系統取代情緒**」
- [ ] 用「**選擇權**」語言取代「**翻身**」語言
- [ ] 讓更多人找到自己的交易/生命邏輯
