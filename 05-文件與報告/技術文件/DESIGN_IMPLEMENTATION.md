## 🚀 如何使用新設計系統

### 1. 在新頁面中應用主題

```python
import streamlit as st
import sys
from pathlib import Path

# 導入主題
sys.path.append(str(Path(__file__).parent.parent))
from utils.ui_theme import apply_theme, format_metric, create_header

# 應用主題
apply_theme(st)

# 創建標題
st.markdown(create_header("📈 頁面標題", "副標題"), unsafe_allow_html=True)

# 顯示指標
st.markdown(format_metric("標籤", "100%", color="#5B47D9"), unsafe_allow_html=True)
```

### 2. 使用快速函數

```python
from utils.ui_theme import (
    apply_theme,           # 應用完整主題
    format_metric,         # 格式化指標卡片
    create_header,         # 創建標題區域
    create_info_box,       # 創建信息框
    get_stat_color         # 根據數值返回顏色
)
```

### 3. 自定義顏色

```python
from utils.ui_theme import (
    PRIMARY_GRADIENT,      # 主色漸變
    SECONDARY_LIGHT,       # 輔助青色
    ACCENT_PINK,           # 強調粉色
    SUCCESS_COLOR,         # 成功綠
    WARNING_COLOR,         # 警告黃
    ERROR_COLOR            # 錯誤紅
)
```

## 💡 使用建議

### 最佳實踐
1. **一致性**: 始終使用 `utils.ui_theme` 中的顏色
2. **可讀性**: 優先使用深文字 (#1F2937) 在淺背景上
3. **層次感**: 使用不同陰影級別創造深度
4. **交互**: 所有可互動元素需要明確的悬停状態

### 常見問題

**Q: 如何改變主色?**
A: 編輯 `utils/ui_theme.py` 中的 `PRIMARY_GRADIENT` 變數

**Q: 深色模式支持?**
