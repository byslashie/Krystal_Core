## 📊 數據來源

所有數據由 Flask 後端通過 API 提供：
- `/api/metrics` - 關鍵指標
- `/api/chart-data` - 圖表數據
- `/api/holdings` - 持倉列表

數據使用 NumPy 和 Pandas 在後端生成和計算。

## 🔄 實時更新

應用每 30 秒自動更新：
- 指標數據
- 持倉信息

圖表在頁面首次加載時渲染。

## 🔧 基本定制

### 改變顏色

編輯 `static/css/style.css`：

```css
:root {
    --primary: #6B21A8;           /* 改這行 */
    --success: #10B981;            /* 或這行 */
    --bg-primary: #F5F0FF;         /* 或這行 */
}
```

### 改變數據

編輯 `app_html_flask.py`：

```python
# 改這個值
total_value = 125345.01  → 改成你的值
holdings_count = 4       → 改成你的持倉數
```

## ❓ 常見問題

**Q: 頁面打不開？**
A: 確保 Flask 應用正在運行（終端中沒有報錯），然後刷新瀏覽器。

**Q: 圖表為什麼是空的？**
A: 等待 3-5 秒讓 JavaScript 加載 Plotly.js。查看瀏覽器控制台看是否有錯誤。

**Q: 怎樣修改色彩？**
A: 編輯 `static/css/style.css` 中的 `:root` 部分。

**Q: 怎樣改變數據？**
A: 編輯 `app_html_flask.py` 中的硬編碼值或連接數據庫。
