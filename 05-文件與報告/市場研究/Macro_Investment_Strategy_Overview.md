# 【總經投資入門攻略】市場關鍵資訊與 Krystal AI 系統實作手冊

本文件深入解析總體經濟與製造業循環指標，並提供將此邏輯部署至 **Krystal AI 交易系統** 的具體規格與參數設定。

## 1. 總經基本面核心：各國經濟體質分析與資金流向
系統中的「長期國別資金配置模組（Country Allocation）」可根據以下特性進行長線部位過濾：
*   **美國 (全球資金避風港 / 消費主導, GDP佔比 67.7%)**
    *   **邏輯**：服務消費是支撐美股的基石。只要「就業穩健 + 移民人口持續遷入」，內需便不會輕易衰退。
    *   **監控指標**：非農就業服務業細項、密大消費者信心指數、核心 PCE。
*   **台灣 / 德國 (景氣超級放大器 / 出口主導)**
    *   **邏輯**：高度仰賴中美終端需求。具備「漲時超漲、跌時超跌」的 3-4 年週期性。
    *   **監控指標**：台灣電子零組件出口年增率、台灣外銷訂單。
*   **中國 / 印度 (新興市場資金池)**
    *   **邏輯**：中國面臨產能過剩，靠政策驅動；印度受惠 2063 年前的人口紅利，為長線做多標的。

## 2. 核心驅動：製造業循環 4 階段與 ETF 配置矩陣
Krystal AI 應建置「全球製造業循環燈號（Global Manufacturing Cycle Signal）」，以 3-4 年為週期進行資產的大類輪動（Asset Rotation）。

*主要訊號來源：**PMI 新訂單 (New Orders)** 減去 **PMI 客戶庫存 (Inventories)***

### 循環四階段與對應系統操作策略
| 階段 | 狀態 | 判定條件 (量化觸發條件) | Krystal AI 建議資產配置 (ETF 標的) | 避開/減碼標的 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 築底段** | 被動去庫存 | `新訂單 > 50` 且 `庫存趨勢向下` | **積極偏多**：股 > 債。<br>科技股 (QQQ)、費半 (SOXX)、台股 (EWT) | 避險性公債 (TLT) |
| **2. 強升段** | 主動補庫存 | `新訂單 > 50` 且 `庫存向上爬升 (供不應求)` | **全面爆發**：全球股市 (VT)、新興市場 (VWO)、大宗商品/銅 (CPER) | 美元指數 (UUP) |
| **3. 末升段** | 被動補庫存 | `新訂單趨緩向下 (<50)` 且 `庫存向上 (滯銷)` | **防禦準備**：債 >= 股。<br>防禦型類股 (XLP)、醫療保健 (XLV) | 週期性板塊 (XLY)、半導體 |
| **4. 衰退段** | 主動去庫存 | `新訂單 < 50` 且 `庫存急劇砍單向下` | **極度防禦**：大量現金、美長債 (TLT / IEF)、美元指數 (UUP) | 新興市場、高收益債 (HYG) |

> **開發實作細節**：
> 中國 PMI 為領先指標，美國 ISM PMI 為確認指標，台灣出口數據為驗證指標。系統邏輯應設定：`IF 中國PMI新訂單 連續 2 個月向上 THEN 觸發台灣電子股 Watchlist 買進籌備訊號。`

## 3. 每月必看美國經濟數據清單與 Krystal AI 追蹤設定
建議透過 FRED API 或 MacroMicro API 自動將以下資料爬取至 Krystal AI 的資料庫：

### A. 就業板塊 (Employment) - 防禦護城河
*   **薩姆規則 (Sahm Rule)**：
    *   *公式*：失業率 3 個月移動平均值 - 過去 12 個月失業率低點。
    *   *系統動作*：當數值 `> 0.5` 時，觸發【經濟衰退警報】，系統強制平倉 30% 以上的高風險曝險並轉入美元與短債 (SHV)。
*   **初領/續領失業救濟金 (每週四高頻資料)**：
    *   *系統動作*：若四週移動均線連續 3 週大幅攀升，降低系統最大槓桿倍數。

### B. 消費板塊 (Consumption) - GDP 成長引擎
*   **零售銷售 (Retail Sales) & PCE**：
    *   *處理邏輯*：計算時排除「汽車」與「汽油燃料」等高波動項目，專注於「核心類別」與「服務類別」。
    *   *前瞻指標*：串接「紅皮書同店銷售指數」(Redbook YoY)，每週提前預判零售強度。

### C. 通膨與資金流向 (Inflation & Fed)
*   **聯準會利率路徑預期 (FedWatch Tool)**：
    *   當聯準會釋出「降息預期區間」時，美國與他國的「利差」縮小，將導致美元走弱。系統應在確認降息循環後，放大對新興市場與風險資產的配置權重。

## 4. 系統開發整合建議 (Action Items)
1.  **資料源介接**：優先對接 `FRED API` 下載美國核心指標，撰寫 Python 後端排程化腳本 (CronJob) 每週五更新數據。
2.  **儀表板 (Dashboard) 升級**：在 Krystal AI 前端新增 **Macro Regime 儀表板**，將當前所處的「製造業循環 4 階段」以燈號顯示（紅/黃/綠燈）。
3.  **停損停利動態調整**：將總經指標掛鉤到現有的跟蹤停損 (Trailing Stop) 參數上。處於衰退段時，所有的停損 % 應緊縮；處於強升段時，停損 % 應放寬以換取波段獲利。
