| # | 列名 | 資料型態 | 說明 | 範例 |

| 7 | 進場數量 | 整數 | 進場股數 | 100 |

| # | 列名 | 資料型態 | 說明 | 範例 |
|----|------|--------|------|------|

| 9 | 出場價格 | 數字 | 出場成交價 | 465.50 |
| 10 | 出場數量 | 整數 | 實際出場股數 | 100 |

### 持倉天數（1 列）
| # | 列名 | 資料型態 | 說明 | 範例 |
|----|------|--------|------|------|
| 11 | 持倉天數 | 整數 | 進場到出場的交易日數 | 6 |

### 計算 P&L（5 列）
| # | 列名 | 資料型態 | 說明 | 計算公式 | 範例 |
|----|------|--------|------|--------|------|
| 12 | 毛利 | 數字 | 未扣費的獲利 | (出場價 - 進場價) × 數量 | 1550 |

| # | 列名 | 資料型態 | 說明 | 範例 |

## 🧮 計算規則（必須遵循）

### 1. 進場成本計算
```
進場成本 = 進場價格 × 進場數量 × (1 + 手續費率)

台股手續費率：0.001425% (元大)
美股手續費率：0.001% - 0.01% (依券商)
```

### 2. 毛利計算
```
毛利 = (出場價 - 進場價) × 出場數量
```

### 3. 手續費計算
```
買進手續費 = 進場價 × 進場數量 × 手續費率
賣出手續費 = 出場價 × 出場數量 × 手續費率
總手續費 = 買進手續費 + 賣出手續費

台股例：
  買進費用 = 450 × 100 × 0.001425% = 0.64 元
  賣出費用 = 465.5 × 100 × 0.001425% = 0.66 元
  總費用 = 1.30 元
```

### 4. 稅費計算（僅台股賣出）
```
稅費 = 出場價 × 出場數量 × 稅率

台股賣出稅率：0.3%
美股賣出稅率：0%

台股例：
  稅費 = 465.5 × 100 × 0.3% = 139.65 元
```

### 5. 淨利 (P&L) 計算
```
P&L = 毛利 - 買進手續費 - 賣出手續費 - 稅費

範例（台股）：
  毛利 = (465.5 - 450) × 100 = 1,550 元
  手續費 = 1.30 元
  稅費 = 139.65 元
  P&L = 1,550 - 1.30 - 139.65 = 1,409.05 元
```

### 6. P&L % 計算
```
P&L % = P&L / (進場價 × 進場數量) × 100%

範例：
  P&L % = 1,409.05 / (450 × 100) × 100% = 3.13%
```

### 7. 累積 P&L（多批次）
```
當有多個批次時：
  累積 P&L = Σ(各批次淨利)
  累積投入 = Σ(各批次進場成本)
  累積 P&L % = 累積 P&L / 累積投入 × 100%

範例（2個批次）：
  第1批：進場成本 45,000, P&L = 1,280
  第2批：進場成本 44,800, P&L = 1,520
  累積投入 = 89,800
  累積 P&L = 2,800
  累積 P&L % = 2,800 / 89,800 = 3.12%
```

## 📥 Excel 模板設置

### 列寬度（建議）
| 列 | 寬度 | 用途 |
|----|-----|------|
| A-D | 15 | 基本信息 |
| E-K | 18 | 日期時間、價格、數量 |
| L-Q | 12 | 計算結果（數字）|
| R | 20 | 出場原因（備註）|

### 格式設置
```
列 F, I：數字格式，2位小數 (450.50)
列 G, J, K：整數格式 (100)
列 N：百分比格式，4位小數 (2.84%)
列 L, M, O：數字格式，2位小數 (1550.00)
```

### 驗證規則
```
列 E, H：日期格式，必填
列 F, I：數字 > 0，必填
列 G, J, K：正整數，必填
列 N, P：百分比 -100% 至 +1000%，必填
列 R：下拉選單，固定選項
```

## ✅ 數據驗證檢查清單

導入前檢查：

- [ ] **日期格式** — 全部使用 YYYY/MM/DD HH:MM
- [ ] **價格** — 2位小數，無千位分號
- [ ] **數量** — 整數，無小數點
- [ ] **持倉天數** — 計算無誤 (exit_date - entry_date)
- [ ] **毛利** — (exit_price - entry_price) × quantity
- [ ] **手續費** — 台股 0.001425%, 美股參考券商費率
- [ ] **稅費** — 台股 0.3%, 美股 0%
- [ ] **P&L** — 毛利 - 手續費 - 稅費
- [ ] **P&L %** — P&L / (進場價 × 數量)
- [ ] **累積 P&L** — 同股票所有批次累計
- [ ] **累積 P&L %** — 累積 P&L / 累積投入
- [ ] **出場原因** — 不空白，分類一致

## 🔄 多幣種處理

### 台股 (TWD)
```
預設無需轉換
所有計算直接用 TWD
```

### 美股 (USD)
```
進場/出場價：直接輸入美元
匯率轉換：報告時按日期匯率轉 TWD
損益計算：保持 USD，報告時轉 TWD
```

### 混合持倉
```
若同時有台股和美股：
- 各自分別計算 P&L
- 報告時統一轉為 TWD（使用該交易日匯率）
- 計算整體回報率時加權平均
```

## 📌 常見錯誤修正

### 1. 日期格式錯誤
```
❌ 錯誤：6/1/2026, 2026-04-01, 2026/4/1
✅ 正確：2026/04/01 09:30
```

### 2. 手續費計算錯誤
```
❌ 錯誤：只計算單向手續費
✅ 正確：買進 + 賣出 + 稅費 都計入

❌ 錯誤：450 × 100 × 0.1% = 450 元
✅ 正確：450 × 100 × 0.001425% = 0.64 元
```

### 3. P&L % 計算錯誤
```
❌ 錯誤：(465.5 - 450) / 465.5 × 100% = 3.33%
✅ 正確：(465.5 - 450) / 450 × 100% = 3.44%
        (使用進場價作為分母)
```

### 4. 多批次累積錯誤
```
❌ 錯誤：第2批次的 P&L % = P&L / (第2批進場成本)
✅ 正確：累積 P&L % = 累積 P&L / 累積進場成本
```

### 5. 稅費計算錯誤
```
❌ 錯誤：只有賣出時計稅，稅率 0.1%
✅ 正確：台股賣出時計稅，稅率 0.3%
        美股不計稅
```
