# Summary
- 入場：箱體上沿突破 + 放量；出場：ATR/KAMA；濾網：N3 與 M2 事件。

✅ 進場：
每月月初（跨月當日），若

標的在選股清單內 (IslistedSymbol = true)，

VIX 濾網通過 (vix(VixLookback) = 0)，

（可選）股價站上 KAMA（多頭趨勢），
則進場建立多單（SetPosition(Money / Close)）。

✅ 出場：
1️⃣ 初始停損：跌破進場價 -Loss_Percent%（例：-8%）出場。
2️⃣ 保本 (BE)：獲利達 +30% 後，若回落至進場價出場。
3️⃣ 第一段停利：獲利達 +70%，回落 20% 出場。
4️⃣ 第二段停利：獲利達 +100%，回落 30% 出場。
5️⃣ 時間停損：持有超過 N 天（例：30天）仍未達 +N%（例：5%）出場。
　　※ 若最高漲幅已達 BigWinPct（例：+50%）則不啟用時間停損。
6️⃣ 清單移除：跨月後不在選股名單內出場。
7️⃣ VIX 危險區：若 VIX 風險過高（vix=1）即時出場。
