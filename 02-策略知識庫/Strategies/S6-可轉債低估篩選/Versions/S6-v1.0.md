# S6 可轉債低估篩選 – v1.0

## 📝 Summary
本版本專為「大盤大跌」時期的抄底設計。核心邏輯在於鎖定 **價格接近 100 且溢價率低於 30%** 的標的，以獲得最強的下檔支撐與上行彈力。

## ⚙️ 策略參數 (Logic)

### 1. 進場篩選 (Entry)
- **CB 價格**: $P_{CB} \in [100, 110]$ 元。
- **到期殖利率 (YTM)**: $> 1.0\%$。
- **轉換溢價率 (Premium)**: $< 30\%$。
- **大盤條件**: 大盤指數收盤價 < $60MA$ (季線) 或 $RSI_{14} < 40$。
- **信用過濾**: TCRI 評等須在 1-6 之間（排除高違約風險者）。

### 2. 出場邏輯 (Exit)
- **停利**: 當 $P_{CB} \ge 130$ 或溢價率反彈至 $> 60\%$ (股性過強)。
- **停損**: 本策略不設價格停損（因有債底保護），但在公司信用評等遭調降至 TCRI 8 以上時強制出場。
- **時間**: 若持有超過 1.5 年且股價無起色，評估是否換倉。

## 📊 預期表現分析
- **優點**: 在空頭市場中極度耐震，適合大資金停泊。
- **缺點**: 若大盤持續陰跌且成交量萎縮，資金效率可能較低。
