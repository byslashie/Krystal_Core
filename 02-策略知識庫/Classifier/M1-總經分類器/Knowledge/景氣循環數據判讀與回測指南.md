# 📘 景氣循環（Business Cycle）數據判讀與回測方法

本文件整理如何使用 **總經數據** 判讀景氣循環，並將其導入 **量化策略回測**（Regime-based Backtest）。

---

# 📌 1. 景氣循環四階段（定義與指標）

以下四象限可用於策略開關、倉位調整或策略切換：

## 🔹 Recovery（復甦期）

- GDP YoY：低位回升 > 0%
    
- PMI：由 <50 → 升破 50
    
- CPI：溫和上升
    
- 失業率：見頂 → 下滑
    
- 庫存：高位 → 開始下降
    

## 🔹 Growth（擴張期）

- GDP YoY：2%–4% 穩健成長
    
- PMI：> 52
    
- CPI：穩定
    
- 失業率：低且穩定
    
- 庫存：低且穩定
    

## 🔹 Slow Growth（放緩期）

- GDP YoY：下降但 >0%
    
- PMI：跌回 <50
    
- CPI：快速下降
    
- 失業率：轉升
    
- 庫存：回到上升趨勢
    

## 🔹 Recession（衰退期）

- GDP YoY：轉負（<0%）
    
- PMI：<45
    
- CPI：接近 0 或通縮
    
- 失業率：快速上升
    
- 庫存：高檔、需求疲弱

# 📌 2. 景氣分類器（Regime Classifier）流程

## Step 1. 數據標準化

```python
df['pmi_z'] = (df['PMI'] - df['PMI'].rolling(12).mean()) / df['PMI'].rolling(12).std()
df['gdp_z'] = (df['GDP'] - df['GDP'].rolling(12).mean()) / df['GDP'].rolling(12).std()
df['cpi_z'] = (df['CPI'] - df['CPI'].rolling(12).mean()) / df['CPI'].rolling(12).std()
df['unemp_z'] = (df['Unemployment'] - df['Unemployment'].rolling(12).mean()) / df['Unemployment'].rolling(12).std()
```

## Step 2. 加權打分

```python
df['regime_score'] = (
    df['pmi_z'] * 0.35 +
    df['gdp_z'] * 0.25 +
    df['cpi_z'] * 0.20 +
    df['unemp_z'] * (-0.20)
)
```

## Step 3. 四象限分類

```python
def classify(score):
    if score > 1:
        return 'Growth'
    elif score > 0:
        return 'Recovery'
    elif score > -1:
        return 'Slow Growth'
    else:
        return 'Recession'

df['regime'] = df['regime_score'].apply(classify)
```

分類輸出格式：

```
date | regime_score | regime
```

# 📌 3. 景氣 × 策略回測（Regime-Based Backtest）

## 方法一：同一策略不同階段調整倉位

```python
df['stock_position'] = df['regime'].map({
    'Growth': 1.0,
    'Recovery': 0.7,
    'Slow Growth': 0.3,
    'Recession': 0.0
})

df['strategy_ret'] = df['stock_position'] * df['asset_ret']
```

用途：

- ETF 輪動
    
- 強勢股策略
    
- 指數策略

## 方法二：景氣不同策略（策略切換）

```python
if regime == 'Growth':
    run_Momentum()

    run_MeanReversion()

```

用途：

- ETF、股票與債券之間切換
    
- 多策略模型（動能、均值回歸、避險）

# 📌 6. 景氣分類器程式模板（econ_classifier.py）

```python
class EconClassifier:
    def __init__(self, df):
        self.df = df
    
    def calculate_scores(self):
        df = self.df
        df['pmi_z'] = (df['PMI'] - df['PMI'].rolling(12).mean()) / df['PMI'].rolling(12).std()
        df['gdp_z'] = (df['GDP'] - df['GDP'].rolling(12).mean()) / df['GDP'].rolling(12).std()
        df['cpi_z'] = (df['CPI'] - df['CPI'].rolling(12).mean()) / df['CPI'].rolling(12).std()
        df['unemp_z'] = (df['Unemployment'] - df['Unemployment'].rolling(12).mean()) / df['Unemployment'].rolling(12).std()

        df['regime_score'] = (
            df['pmi_z'] * 0.35 +
            df['gdp_z'] * 0.25 +
            df['cpi_z'] * 0.20 +
            df['unemp_z'] * (-0.20)
        )

        df['regime'] = df['regime_score'].apply(self._classify)
        return df[['regime', 'regime_score']]

    @staticmethod
    def _classify(score):
        if score > 1: return 'Growth'
        elif score > 0: return 'Recovery'
        elif score > -1: return 'Slow Growth'
        else: return 'Recession'
```
