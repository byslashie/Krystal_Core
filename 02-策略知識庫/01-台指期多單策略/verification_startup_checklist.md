## 🔍 關鍵觀察點

### 進場條件檢查
進場前務必檢查以下 5 項**都為 ✓**（缺一不進）：

```
✅ 條件2：大盤 < 60EMA × 0.997
   → 判斷：大盤是否走弱（低於 EMA）？
   
✅ 條件3：VIX > 17
   → 判斷：波動率是否足夠高？
   
✅ 條件6：逆價差（期 < 現）
   → 判斷：期現價差是否反向？
   
✅ 條件11：高波動下跌 K
   → 判斷：最近 K 線是否下跌 + 波幅大？
   
✅ 條件10：成交量 > 5 根均
   → 判斷：成交量是否突增？

⚠️ 如果有任何一項 ✗，就停止進場！
```

### 出場信號判斷
出場時記錄是以下哪一種方式：

```
【方式1：高位反轉】 (主要)
  %B > 72 AND %B < 120 + 正價差
  → 記為：「%B高+正價差」
  
【方式2：停損】 (風險管理)
  Close ≤ 進場價 - 200 點
  → 記為：「停損」
  
【方式3：強制平倉】 (時間限制)
  進場日 13:40 或隔日 13:40
  → 記為：「尾盤平倉」或「隔日 13:40 平」
```
