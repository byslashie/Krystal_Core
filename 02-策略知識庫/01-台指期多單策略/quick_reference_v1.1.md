## 🎯 3 秒合規檢查

```
□ 時間在 08:45-13:25？
□ 今日進場 ≤2 次？
□ 不是週五？
→ 都勾才能進場！
```

## 🚀 進場 5 大條件（全部勾才進場）

| 條件 | 檢查 | 符合 |
|---|---|---|
| **條件2** | 大盤 < 60EMA × 0.997 | ✓/✗ |
| **條件3** | VIX > 17 | ✓/✗ |
| **條件6** | 逆價差（期<現） | ✓/✗ |
| **條件11** | 下跌K + 波幅>0.13% | ✓/✗ |
| **條件10** | 成交量 > 5根均 | ✓/✗ |
| | **全部✓才進場** | ✓✓✓✓✓ |

---

## 📤 出場 3 種方式

```
【方式1：高位反轉】
  %B > 72 AND %B < 120 + 正價差
  → 立即出場，鎖住獲利

【方式2：停損】
  Close ≤ 進場價 - 200點
  → 立即出場，止損

【方式3：強制平倉】
  進場日 13:40 / 隔日 13:40
  → 系統自動平倉
```

## ⏰ 關鍵時間點

| 時段 | 動作 | 備註 |
|---|---|---|
| **08:45 開始** | 可進場 | 日盤開始 |
| **13:25 截止** | 禁止進場 | 之後禁止新進場 |
| **13:40 清倉** | 強制平倉 | 必須出場 |
| **13:25-15:00** | 禁止時段 | 不要進場！ |
