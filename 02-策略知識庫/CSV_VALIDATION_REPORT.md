### 問題 2：「獲利金額」是否已包含手續費和稅費？

**當前驗證**：
```
根據我們的計算公式：
  毛利 = (出場價 - 進場價) × 數量
  手續費 = 約 0.001425% × 2（買進+賣出）
  稅費 = 0.3%（賣出）
  淨利 = 毛利 - 手續費 - 稅費

但實際獲利金額與毛利差異太大，可能有以下解釋：
  1. 數量單位不同（見問題1）
  2. 獲利金額已經包含費用調整
  3. 或者使用了不同的計算方式
```

**需要你確認**：
- ✓ 「獲利金額」是否已經扣除手續費和稅費？
- ✓ 還是需要我們重新計算？
