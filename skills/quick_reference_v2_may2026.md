## 🧠 新決策來了？— 8 鏡頭檢查 (3 分鐘)

### 必做 4 個

1. **交易員思維**：投入多少？回報多少？
   - ROI? 時間成本? 機會成本?
   
2. **哲學家思維**：與 2033 年目標一致嗎？
   - 是「逃避」還是「前進」？
   
3. **PM 思維**：有 Bug 嗎？有度量嗎？
   - 邊界清楚嗎? 退出計劃呢?
   
4. **健康框架**：證據強 + 低成本 + 快速驗證？
   - 證據 > 60%? 時間 < 30 分/天? 金錢 < NT$100?

### 視情況 4 個

5. 是自媒體決策 → **痛點 vs 流量陷阱**（真實痛點？3 個 TA 提過？）

6. 是交易決策 → **風險管理**（虧損 2% 以內？連虧 3 停手？）

7. 感到疲勞 → **注意力管理**（在多開視窗嗎？）

8. 所有決策 → **PDCA**（有反饋迴路嗎？2 週檢查？）

## ⚠️ 禁止行為清單

立即停止這些行為：

```
❌ 多開視窗工作
   → AI 跑的時候閉眼休息，別切去做別的

❌ 同時開兩個 AI 角色會話
   → 只用一個綜合 Skill，不要 separate 會話

❌ 在身份間頻繁切換
   → 09-12 交易、13-17 PM、19:30-21 內容，中途不要跳

❌ 用「複雜假設診斷」替代「實驗驗證」
   → 先試 4 週，再決定要不要長期投入

❌ 沒有度量指標就開始做
   → 怎麼知道成功？4 週後評估標準是什麼？

❌ 2026 年內就做「課程」變現
   → 數據還不夠，先積累信任

❌ 分享「未驗證的建議」
   → 所有內容必須基於實驗數據 or 學術文獻
```

```
→ 激活迷走神經副交感分支
```

## 📈 交易快查

### v1.1 進場 5 大條件
```
✅ 時段：08:45-13:25 內
✅ 大盤 < 60EMA × 0.997
✅ VIX > 17
✅ 逆價差（期 < 現）
✅ 成交量 > 5 根均 + 高波動下跌 K

→ 缺一不進！
```

### 禁止日期
```
❌ 週五禁止進場（所有時段）
❌ 13:25-15:00 禁止進場
```

### 出場方式
```
1. 高位反轉：%B > 72 AND %B < 120 + 正價差
2. 停損：Close ≤ 進場價 - 200 點
3. 尾盤平倉：進場日 13:40 or 隔日 13:40
```
