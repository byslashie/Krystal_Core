## 📊 四重身份系統

### 優先級排列（P0 → P3）

```
P0 - 人生管理 & 健康（10%）
  ├─ 目標：實驗→測試→評估→決策，砍掉無效努力
  └─ 時段：22:00-23:00（日誌 + 自我反思）

P1 - 交易機會 & 市場監控（40%）
  ├─ 目標：日盤核心 + 合規性修復 + ATR波動判斷
  └─ 時段：09:00-12:00, 17:00-18:00, 21:30-23:00

P2 - PM 項目交付（35%）
  ├─ 目標：系統化、可交付、有度量
  └─ 時段：13:00-17:00（集中工作）

P3 - 自媒體 / 財經 KOL（5%）
  ├─ 目標：內容支柱 + 痛點驅動 + 機會成本評估
```

- 健康警示（過度疲勞）→ 立即停止休息

## 🔭 八個思維鏡頭

每次做決策時，按順序審視：

### 鏡頭 1：交易員思維 — 成本效益
```
問題：投入多少？回報多少？
方法：ROI、Kelly 公式、蒙特卡洛模擬
檢查清單：
  □ 時間成本：多久才能驗證？
  □ 金錢成本：比月收入的百分比？
  □ 機會成本：放棄什麼更大的機會？
```

### 鏡頭 2：哲學家思維 — 價值對齐
```
問題：這與我的長期目標一致嗎？
方法：以終為始、反向工程、價值檢驗
檢查清單：
  □ 2033 年的目標支持嗎？
  □ 違反了什麼原則嗎？
  □ 這是「逃避」還是「前進」？
```

### 鏡頭 3：PM 思維 — 系統性檢查
```
問題：這個方案有 Bug 嗎？有隱藏的成本嗎？
方法：PRD、拆解、風險矩陣
檢查清單：
  □ 邊界清晰嗎（什麼時候開始、何時結束）？
  □ 有度量指標嗎（怎麼知道成功了）？
  □ 有退出計劃嗎（失敗時怎麼辦）？
```

### 鏡頭 4：健康決策框架 — 成本優先順序
```
問題：這符合「證據強度 + 低成本 + 快速驗證」嗎？
方法：實驗框架（4 週驗證計畫）
檢查清單：
  □ 證據強度 > 60%？
  □ 時間成本 < 30 分鐘/天？
  □ 金錢成本 < NT$100？
  □ 3-7 天內能見效嗎？
```

### 鏡頭 5：痛點 vs 流量陷阱（自媒體）
```
問題：這是真實痛點還是流量陷阱？
方法：開放式編碼、TA 訪談驗證
檢查清單：
  □ 至少 3 個 TA 提過這個問題？
  □ 他們願意花錢解決嗎？
  □ 我能獨特地解決嗎？
```

### 鏡頭 6：風險管理 — 連虧停手
```
問題：最壞情況是什麼？我能承受嗎？
方法：連虧停手規則、風險敞口限制
檢查清單：
  □ 最大虧損 = 帳戶 2% 以內？
  □ 連虧 3 次停手？
  □ 有自動平倉機制嗎？
```

### 鏡頭 7：注意力管理 — 單一上下文
```
問題：我在製造「注意力殘留」嗎？
方法：時間塊狀化、禁止多開視窗
檢查清單：
  □ 在一個身份的時段內工作？
  □ 沒有頻繁在 App/工作間切換？
  □ AI 跑的時候，眼睛閉著休息？
```

### 鏡頭 8：PDCA 迭代循環
```
問題：這個決策有反饋迴路嗎？
方法：Plan → Do → Check → Act，2 週一次
檢查清單：
  □ 計劃（P）：目標 + 預期 + 驗證方法
  □ 執行（D）：按計劃執行，記錄數據
  □ 檢查（C）：評估是否達成
  □ 調整（A）：基於數據做下一步決策
```

## 🎬 自媒體內容三大支柱

### 支柱 1：交易系統分享
- **內容**：台指期策略、籌碼分析、技術指標
- **TA 痛點**：「怎麼穩定獲利？」「怎麼管理風險？」
- **形式**：策略拆解、實盤日誌、回測數據
- **變現**：課程（2027 Q1 後）

### 支柱 2：人生規劃實錄
- **內容**：四重身份時間管理、決策框架、實驗設計
- **TA 痛點**：「怎麼在多個身份間平衡？」「怎麼快速驗證想法？」
- **形式**：週反思、月總結、案例分析
- **變現**：諮詢（2027 Q2 後）

### 支柱 3：身心平衡實驗
- **內容**：低 FODMAP 飲食、迷走神經激活、自律神經調節
- **TA 痛點**：「怎麼改善腸胃問題？」「怎麼減少疲勞？」
- **形式**：4 週驗證日誌、數據對比、方案決策
- **變現**：線上社群（2026 Q4 後）

## ✅ 使用流程

### 面對新決策時（3 分鐘檢查）

```
1. 鏡頭 1 & 3：成本效益 + 系統性檢查
   → 投入多少？有 Bug 嗎？

2. 鏡頭 2 & 8：價值對齐 + PDCA
   → 與長期目標一致嗎？有反饋迴路嗎？

3. 其他鏡頭：根據身份和情境選用
   → 是自媒體決策？用鏡頭 5
   → 是交易決策？用鏡頭 6
   → 是健康決策？用鏡頭 4

4. 鏡頭 7：注意力檢查
   → 我在多開視窗嗎？
```

### 面對複雜問題時（15 分鐘深度分析）

```
按順序走過 8 個鏡頭
記錄每個鏡頭的發現和建議
綜合得出最終決策
設計 PDCA 驗證計畫
```
